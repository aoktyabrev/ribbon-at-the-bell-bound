## Теория (слои)
1. Онтология (Артема): запутанная пара = один объект («лента»), два конца —
   границы в 3D-срезе объекта, вложенного в большую размерность (4D).
   Вложение в 4D даёт ℤ₂-классы оснащения (π₁(SO(3))=ℤ₂) — происхождение
   и бинарности исходов, и спинорности (720°).
2. Математический слой: закон хорды P(s,t|a,b) = |s·a − t·b|²/8 ⇒
   E = −cosθ, cos²(θ/2), no-signaling, независимость порядка.
3. Выведено (сессии 1–3, код в repo): полуугол — из S³/трюк с ремнём;
   антикорреляция — из нулевой скрутки; квадрат (p=2 в семействе
   P ∝ |sa−tb|^p) — ЕДИНСТВЕННОСТЬ ИЗ ПРИЧИННОСТИ: любое отклонение
   (p≠2, ε≠0) даёт телеграф на частично запутанной связи (Δ=0.146 при
   p=1, 0.081 при p=3, 0.072 при ε=0.2; переоткрытие Жизана-1990).
   «Калиброванная непрозрачность»: борновская мера зажата с ОБЕИХ сторон.
4. Отрицательные результаты (контрпримеры): согласованность симметричной
   пары НЕ фиксирует Борна (ε-семейство проходит всю батарею);
   равномерная мера + конусные бассейны дают пилу (классика);
   «закон сохранения постулата» (правило ↔ мера ↔ веса);
   Кохен–Спекер-1967 = доказательство существования меры (постулат
   переезжает в меру). Семейство хорды: p=1 CHSH 1.66, p=2 = КМ 2.83,
   p→∞ = PR-box 4.0.

## Симуляция ribbon-sim (фазы A/B/C ЗАКРЫТЫ, все зелёные)
Цепочка кватернионов на S³, зажимы −k_c(n·a)², ланжевен/PT.
Ключевые результаты:
- Лента — ферромагнетик (E(0)=+1); изотропное ферро E=+a·b квантово
  нереализуемо (σxσx·σyσy·σzσz=−I). Конвенция: глобальный флип t̃=−t.
- «Ножницы амплитуда–форма»: |E(0)|=1 и гладкий косинус несовместимы
  в классической релаксации.
- No-go гейдж-слепоты: осевые наблюдаемые (функции n=R(q)e_z)
  факторизуются через SU(2)→SO(3), слепы к ℤ₂; кинки без топозащиты
  уходят, термически не населяются (E_kink ~ k_e·π² ≫ T).
- Динамический интерьер: бассейны определяются динамикой, не ландшафтом
  (амплитуда 0.66(N=2) → 0.38(N≥16), насыщение).
- ГЛАВНОЕ (R4b-PT/R4c/R4d): топологический заряд Tw=2π (нетривиальный
  ℤ₂-класс) переворачивает знак и рождает СИНГЛЕТНУЮ ФОРМУ через
  энтропийную меру: E(θ|2π) = −tanh(ΔS·cosθ/2), ΔS≈1.1·cosθ, ΔE≈0 —
  вероятность ветви = подсчёт микроконфигураций. Косинус РОЖДЁН, не вложен
  (сверка с эталонами была заблокирована до фиксации сырых весов).
  Заряженный сектор термически недостижим — только приготовление
  (рифма: синглет готовится источником).
- НО: A≈0.5, CHSH≈1.48<2. R4e-v2: носитель заряда — локализованный
  твист-солитон; кроссовер ферро→антиферро→0 по плотности твиста
  2π/(N−1); эффект конечно-размерный, гаснет при N→∞.
- ЛЕММА (финал): классической энтропийной мере не хватает двух квантовых
  инвариантностей: (1) ТОЧНЫЕ НУЛИ (Борну нужен запрет исхода/
  суперселекция, энтропия даёт лишь штрафы; A=tanh(ΔS/2)<1 всегда);
  (2) МАСШТАБНАЯ ИНВАРИАНТНОСТЬ (квантовая корреляция не разбавляется
  расстоянием, солитонный механизм — разбавляется).

## ФАЗА D (текущая задача — спроектировать)
Вопрос: может ли топология ВЛОЖЕНИЯ (двумерная поверхность в R⁴,
3D-срез, две точки пересечения = «частицы», зажимы в срезе, классы
оснащения ℤ₂) дать (1) геометрический ЗАПРЕТ выровненной ветви
(нулевая мера по построению) и (2) инвариантность к растяжению.
Плюс PBR-тест на любую будущую меру (носители мер разных состояний
не пересекаются). Нужна новая дискретизация (поверхность, не цепочка).
Ожидания калибровать честно: вероятный исход — очередные именованные
стены; это тоже ценность.

## Методология (обязательна)
Пре-регистрация до запуска (не редактируется после); блочная сходимость
по наблюдаемой (не по энергии); зеркальные пары; честный сэмплер секторов;
PT для глассовых ландшафтов (адаптивная лестница, RT≥3, ладдер-
независимость); класс DEGENERATE для вырожденных основных состояний;
сверка с косинус-эталонами ТОЛЬКО после фиксации сырых весов; аудит
любого «слишком хорошего» результата (прецеденты: транзиент «p≈2»,
ложные межсекторные различия на недосходимости). Журнал исправленных
гипотез архитектора: пила при κ=10 (отозвана до данных), float32-диагноз
R3, энергетический tanh R4d, коммензурабельность R4e. Система аудита
бьёт по обе стороны стола — поддерживать.
