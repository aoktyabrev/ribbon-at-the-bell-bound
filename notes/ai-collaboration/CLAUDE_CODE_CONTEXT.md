  phase-C-final-v2 (439516c) + финальные документы архитектора в корне

  недостижим. Лемма в phase_C_report.md §7.

## Валидированный инструментарий (не ломать, переиспользовать)
- energy.py: geodesic | spinor | cosserat_geo (жёсткая анизотропия,
  побитово-симметричная; atan2-Коссера DEPRECATED c известным дефектом,
  xfail-тест); chordal — мягкий, только для сравнений.
- sector_sample: честная выборка сектора Tw=target (Haar-разнообразие,
  err<1e-6); Tw = Σ 2z_i, проекция градиента + Ньютонова коррекция.
- pt.py: параллельное темперирование (адаптивная лестница по равному
  acceptance, свопы каждые 10 шагов, диагностика RT/ладдер-независимость).
  Урок: энергошкалу с N мостить ЧИСЛОМ ступеней, не подъёмом T_hot.
- Зеркальные пары (q, q⊗j) — контроль |E_h1−E_h2|; глобальный флип
  конвенции t̃=−t; блочная сходимость по наблюдаемой; классификация
  DEGENERATE (сид-флип при честном сэмплере = вырожденность, не баг);
  fit_shapes (MLE+AIC: синглет/хорда/ступень/пила), гармоники A1/A3;
  диагностика меры (гистограммы c_A,c_B по ветвям), профиль ⟨τ̃_i⟩.
- Правило: сверка форм с эталонами только ПОСЛЕ фиксации сырых весов.

## Известные грабли (не наступать повторно)
- Транзиенты неполной релаксации выглядят как открытия (прецеденты:
  «p≈2» R1, немонотонность R4a, межсекторные различия на 8k шагов).
  Любой интересный результат → аудит сходимости до интерпретации.
- Осевые наблюдаемые гейдж-слепы к ℤ₂ (no-go); кинки без топозащиты
  уходят; заряженный сектор достижим только приготовлением.
- Пер-линковые жёсткости при свипах N держать константными (материал
  один); k∝N даёт энергию ∝N² и ломает эквилибрацию.
- fp64 на RTX 4070 Ti ~1/64 скорости — только мини-ячейки, с
  инкрементальной записью результатов.
- GPU может быть занят посторонним python3 main.py (~7 ГБ, тренировка
  NEX) — проверять окно перед тяжёлыми прогонами;
  XLA_PYTHON_CLIENT_PREALLOCATE=false уже стоит.

## Что дальше — ФАЗА D (НЕ НАЧИНАТЬ без SPEC-D)
Архитектор проектирует фазу D: дискретизация двумерной поверхности,
вложенной в R⁴; 3D-срез с двумя пересечениями («частицы»); зажимы
действуют в срезе; классы оснащения ℤ₂. Два пре-регистрируемых вопроса:
(1) геометрический запрет выровненной ветви (нулевая мера по построению
вместо энтропийного штрафа); (2) инвариантность эффекта заряда к
растяжению. Это НОВАЯ дискретизация — не переделка цепочки; жди
SPEC-D.md от архитектора. До его получения легальны только: поддержка
инструментария, опциональная запаркованная задача (доперемешивание
N−1=40–56, низкий приоритет), ничего нового по науке.
