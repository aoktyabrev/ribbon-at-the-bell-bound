# D_synthesis_S1 — выводима ли A∞≈0.363 из подсчёта бассейнов?

- Дата: 2026-07-14. Аналитическая задача, БЕЗ прогонов. Данные: D1 (dynamics_full),
  D2 (raw), D2-ext (raw+analysis), census_full.
- Вопрос: выводима ли амплитуда плато A∞≈0.363 (нечёт, θ=0) из подсчёта совместных
  бассейнов исходов (s,t)?

## 1. Эмпирика бассейнов (θ=0, нечётный сектор)

Совместная доля выровненных исходов P_aligned = (n₊₊+n₋₋)/n_valid (DEGENERATE=0 во
всех точках ⇒ n_valid=M, арифметика чистая):

| источник | N | M | P_aligned | E(0)=2P_al−1 |
|---|---|---|---|---|
| D1 | 16 | 200 | 0.65–0.69 | +0.30…+0.38 |
| D1 | 48 | 200 | 0.64–0.70 | +0.28…+0.40 |
| D2-ext | 16..96 | 1200 | ≈0.681 | +0.362 (плато) |

**Тождество (не вывод!):** при θ=0 и знаковой свёртке E=⟨s·t⟩,
  **A∞ = |E(0)| = P_aligned − P_anti = 2·P_aligned − 1** (DEGENERATE=0).
Т.е. A∞ ЕСТЬ асимметрия бассейнов по построению. «Подсчёт бассейнов» при θ=0 не
ВЫВОДИТ A∞ — он ТОЖДЕСТВЕНЕН ему. Содержательный вопрос: **выводима ли
P_aligned≈0.681?** (обратно: A∞=2·0.6815−1=0.363 ✓, tanh(0.38)=0.3627 ✓ — согласовано).

## 2. Природа расщепления 68/32: ЭНТРОПИЯ, не энергия

Census (T=0 минимумы бассейнов, нечёт θ0):

| ветвь | E_min | тип |
|---|---|---|
| ++ | −3.889 | aligned |
| −− | −3.668 | aligned |
| +− | −3.785 | anti |
| −+ | −3.784 | anti |

Среднее выровненных ⟨E_al⟩ = −3.779; анти ⟨E_anti⟩ = −3.785. **Почти ВЫРОЖДЕНЫ**
(ΔE ≈ 0.006 ≪ T=0.05). При Больцмане это дало бы P_aligned≈0.5, а не 0.68.
⇒ расщепление 68/32 — **ЭНТРОПИЙНОЕ** (объём/область притяжения бассейна), НЕ
энергетическое. Согласуется с ПЛОСКОСТЬЮ по T (D1: P_aligned 0.65/0.69/0.69 при
T=0.02/0.05/0.10 — почти не зависит от T, как и положено чисто мерному эффекту).
(Внутривыровненная асимметрия ++/−− по E_min — артефакт старта интерьера в +ê, D1 §; на
СУММУ P_aligned не влияет.)

Следствие: A∞ — не термодинамическое число, а **геометрическая мера области
притяжения выровненного бассейна** при равномерном приготовлении n_end на S²×S².

## 3. Соответствие с энтропийной мерой R4b (фаза C): ОДНО число, не два

- Фаза C: E(θ|2π) = −tanh(ΔS·cosθ/2), ΔS≈1.1 ⇒ |E(0)|=tanh(0.55)=0.50.
- Фаза D (фит D2): E(θ) = −tanh(β·cosθ), β≈0.38 ⇒ |E(0)|=tanh(0.38)=0.363.
- Если один механизм: β = ΔS_eff/2 ⇒ **ΔS_eff = 0.76** (D) vs ΔS = 1.1 (C).

| параметризация | фаза C | фаза D (осевое) |
|---|---|---|
| амплитуда \|E(0)\| | 0.50 | 0.363 |
| ΔS (эфф.) | 1.1 | 0.76 |
| β=ΔS/2 | 0.55 | 0.38 |
| P_aligned=(1+\|E\|)/2 | 0.75 | 0.681 |

**Это ОДНО число** (амплитуда), выраженное четырьмя способами; не два независимых.
ΔS_eff(D)/ΔS(C) = 0.76/1.1 ≈ **0.69** — осевое считывание видит ~69% полной энтропийной
структуры (остаток — в ℤ₂-слое, слепом для оси; U(1)-теорема D1). [Числовое совпадение
0.69 ≈ P_aligned=0.68 — ОТМЕТИТЬ как возможную связь ИЛИ совпадение; не утверждать.]

## 4. Гипотезы происхождения числа (с проверяемыми следствиями)

- **H-a (энтропийный объём бассейна).** P_aligned = доля меры S²×S² (равномерный
  n_A,n_B), стекающая в выровненный бассейн под T=0-потоком. Чисто геометрическое,
  T-независимое число. Следствие: A∞ не зависит от T (✓ D1) и от жёсткостей k_f,k_c
  ТОЛЬКО через форму бассейнов. Проверяемо: карта потока S²×S²→{al,anti} и её мера.
- **H-b (сквозная корреляция репер-цепи, трансфер-матрица).** A∞ = ⟨s_A s_B⟩ —
  наведённая цепью оснащения корреляция двух торцевых «спинов». Плато по N ⇒ эффективная
  корреляция насыщается (зажимы + конечная корреляционная длина). Следствие: A∞ зависит
  от k_f (жёсткость твиста) и слабо от N при N≫ξ. Проверяемо: A∞(k_f).
- **H-c (топологический вычет 2π).** В нечётном секторе обязательный 2π-виток задаёт
  предпочтение выровненности через геометрию транспортированного репера (связь c_i);
  A∞ — «остаточная выровненность» 2π-солитона. Следствие: чётный сектор (без 2π) даёт
  ДРУГОЙ P_aligned. Проверяемо: сравнить P_aligned(чёт) vs (нечёт) при равномерном prep
  (в D1 чёт θ0 aligned≈0.75 vs нечёт≈0.68 — есть разница ~0.07, слабая опора H-c).
- **H-d (численное совпадение).** A∞≈0.363 близко к 1/e=0.3679 (в пределах ~0.4σ).
  Низкий приор; отметить, проверить только если H-a/b/c дадут замкнутую форму рядом с 1/e.

Наиболее вероятно: **A∞ = мера области притяжения (H-a), реализуемая как сквозная
корреляция (H-b)** — две стороны одного (поток = трансфер), с топологической
модуляцией сектора (H-c даёт лишь сдвиг чёт↔нечёт ~0.07).

## 5. Какие ПРОГОНЫ потребовались бы (для решения архитектора; НЕ выполнять сейчас)

## 6. Меняет ли РЕПЕРНО-ЗАЖАТОЕ приготовление подсчёт бассейнов? (вход в go/no-go фазы E)

Текущая D-модель: ОСЕВОЙ зажим −k_c(n_end·a)² фиксирует ОСЬ n_end=±a, оставляя
U(1)-слой (вращение репера вокруг оси) свободным. Именно свободный слой прячет ℤ₂
(U(1)-теорема D1) и даёт мягкий сектор.

**Карандашный аргумент ЗА (реперный зажим МЕНЯЕТ подсчёт):** зажим полного репера
u_end (а не только оси) убирает U(1)-свободу. Тогда ℤ₂-класс лифта становится
ОТЛИЧИМ зажимом (зажать к +u против −u), сектор — clamp-селектируемым, «слепой срез»
снимается. Бассейны выровнено/анти в нечёт расщепились бы ИНАЧЕ (возможно до нулевой
меры одной ветви — ТОЧНЫЙ запрет, дыра 1). Т.е. реперный зажим В ПРИНЦИПЕ меняет
подсчёт и потенциально доставляет суперселекцию.

**Карандашный аргумент ПРОТИВ (это нефизично):** физический аппарат считывает
SO(3)-ОРИЕНТАЦИЮ, а не S³-лифт — это и есть содержание U(1)-теоремы (D1). «Реперный
зажим», фиксирующий знак лифта u vs −u, читает ГЕЙДЖ/спинорную компоненту,
недоступную классическому измерению (тот же нефизичный «лаб-лифт», что в D1-B).
Использовать его = приготовить состояние по НЕнаблюдаемой величине. Подсчёт бассейнов
для ФИЗИЧЕСКОГО (осевого) считывания при этом не меняется — новой физики нет,
изменение чисто в нефизичном датуме.

**Go/no-go для фазы E.** Линия «реперный зажим → точный запрет → дыра 1» — NO-GO,
ЕСЛИ зажим лифта остаётся математическим приёмом (U(1)-теорема блокирует: читаем
недоступное). ЕДИНСТВЕННЫЙ легальный GO: найти ФИЗИЧЕСКУЮ структуру R⁴-вложения,
которая ломает U(1)-слой НАБЛЮДАЕМО — например, вторую фундаментальную форму
вложенной поверхности / преимущественную нормаль из геометрии среза, физически
связанную с фиброй. Тогда ℤ₂ становится измеримым БЕЗ нефизичного лаб-лифта, и
реперный зажим — законное следствие геометрии, а не рука. Крус решения фазы E:
**существует ли в R⁴-вложении физический фибро-ломающий инвариант?** Если да — GO
(дыра 1 достижима геометрически); если нет — NO-GO (U(1)-теорема окончательна,
фаза D закрывает линию считывания топологии, остаётся только «слабый неразбавляемый»
результат D2-ext).

## Резюме

A∞≈0.363 при θ=0 ТОЖДЕСТВЕННО асимметрии бассейнов 2P_aligned−1 (P_aligned≈0.681);
это ОДНО число (= tanh(0.38) = ΔS_eff/2-форма). Расщепление ЭНТРОПИЙНОЕ (бассейны
почти вырождены по энергии, T-независимо). Из первых принципов НЕ выведено; ведущая
гипотеза — геометрическая мера области притяжения (H-a≡H-b). Реперный зажим меняет
подсчёт лишь ценой нефизичного чтения лифта ⇒ фаза E легальна только при наличии
физического фибро-ломающего инварианта R⁴-вложения.
