## PREREG (дословно)

- SA-H1: межсидовый разброс совместим с биномиальным (r ≤ 1.3 по ядру);
  тогда σ фазы D honest, «flat within error» остаётся, в §5.1 добавляется
  одна фраза с числом r.
  Сценарий B (r > 1.3): σ_точки = s_seed по всей фазе D; пересчёт ΔAICc
  плато. B1: ΔAICc(M0 vs M1) ≥ 4 сохраняется → плато стоит, ошибки
  честно раздуты. B2: ΔAICc < 4 → формулировка плато по всей статье
  (вкл. §7.1 «kinetic plateau» и канон) смягчается до «no detected decay
  over N ∈ [16,96] at seed-honest errors; constant and slow-decay models
  not discriminated» — и это вносится ЕДИНОЙ правкой, без попыток спасти
  прежнюю формулировку.
- SA-H2: разброс не растёт с N (s_seed(96) ≤ 1.5·s_seed(32)).
- Калибровка: априори ожидаю r в диапазоне 1.2–1.8 (три точки S1 дали
  выборочное СКО ~0.037 ≈ 1.4·σ_bin); сценарий B вероятен, B2 — возможен.
  Любой исход улучшает статью: либо подтверждённые σ, либо честные.

## Метрики анализа (фиксированы ДО прогонов)

- s_seed = выборочное СКО (ddof=1) по сидам: ядро — 11 точек (8 новых + 3 S1);
  N=96 — 5 точек (4 новых + D2-ext).
- r = s_seed / σ_bin, где σ_bin ≈ 0.026 — медианная биномиальная σ по тем же точкам.
- Переоценка плато: фит M0 (const A∞) / M1 (A·N^−γ) / M2 (A∞+c·N^−γ) по данным D2-ext
  с σ_точки = max(σ_bin, s_seed); ΔAICc(M0 vs M1) — по той же функции aicc(), что в
  analysis_ext.py (без изменений формулы).
- Таблица всех известных повторов одной конфигурации по фазе D той же меркой.
