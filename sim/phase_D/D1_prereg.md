## PREREG (дословно)

- D-H3a (перепись): в нечётном секторе при θ=0 выровненные ветви (s=t) не имеют
  несингулярных представителей (NOT-FOUND по всем стартам + гомотопический
  аргумент); в чётном секторе — зеркально для антивыровненных.
  Kill: EXISTS-представители найдены во всех ячейках → запрет не конструктивный,
  вопрос уходит в меру → высокий риск стены «энтропийный рецидив».
- D-H3b (динамика): частота событий запрещённой ветви = 0 при всех T и обоих N
  на принятых траекториях; rejected_singular →0 при dt→0 независимо от ячейки.
  Kill-1 «энтропийный рецидив»: частота >0 и убывает с T согласованно с
  exp(−c/T) → подавление, не запрет.
  Kill-2 «дырявый лифт-2»: нули достигаются только за счёт роста доли
  отбраковок в запрещённых ячейках → запрет изготовлен фильтром.
- D-H3c (считывание): хотя бы один из кандидатов {A, C, B-loc} проходит тест
  локальности (стабилизация при k≤8) И даёт статистику, отличающуюся от AX
  хотя бы в одной ячейке. Kill «слепой срез»: все локальные кандидаты
  совпадают с AX во всех ячейках → срезовые датумы SO(3)-факторизуемы;
  носитель эффекта — только структура бассейнов (это не отменяет D-H3a/b,
  но закрывает линию «считывание видит топологию»).
- Калибровка: вероятный исход — «слепой срез» по D-H3c при любом исходе
  D-H3a/b; конструктивный запрет по D-H3a — маловероятный сильный результат,
  автоматически попадает под аудит «слишком хорошего».
