# D1 REPORT — считывающий оператор и тест пустого сектора (D-H3)

- Дата: 2026-07-14. Коммит-база: f25694d (phase-D0 green).
- Пре-регистрация: `D1_prereg.md` (D-H3a/b/c), `D1_prereg_addendum.md` (формы §1–§7).
- Окружение: CPU-JAX float64. N∈{16,48}. GPU/NEX не тронуты.
- ЗАПРЕТЫ D1 соблюдены: никаких E(θ) кроме θ∈{0,π}; никаких сверок с косинусами/КМ;
  только СЫРЫЕ (s,t) (конвенционные флипы — не раньше D2).

## 1. Протокол измерения (`measurement.py`)

- Осевой зажим концов −k_c(n_end·a)², n_end=rotmat(u_end)·ê ∈ R³ (норм. коорд),
  ê=(0,0,1). Обе ветви n_end=±a доступны (два минимума). Аппаратные оси a=(0,0,1);
  θ=0 ⇒ b=a; θ=π ⇒ b=−a.
- Секторы: чётный — u≡const (φ≡0); нечётный — 2π-накрутка. Граничные реперы дают
  наблюдаемые (s,t)=(sign(n_A·a), sign(n_B·b)); DEGENERATE при |n_end·a|<0.2.
- **Связь ветвь↔сектор**: осевой зажим фиксирует лишь ОСЬ n_end, оставляя U(1)-слой
  свободным ⇒ ℤ₂-сектор МЯГОК (граничный лифт может доворачиваться). Топо-сектор =
  приготовленный класс при нуле пересечений стены (сохранение по branch-каноническим
  реперам frame_for_axis(s|t) чисто разделяет чёт=+1/нечёт=−1).

## 2. D1-A. Топологическая перепись (`census.py`, `census_notes.md`)

**Численно** (N=48, 24 старта/ячейку, T=0 минимизация, +адиаб. протаскивание):
**все 16 ячеек {сектор}×{(s,t)}×{θ} = EXISTS.** Несингулярные минимумы (rej=0 у
хороших реплик), degen=0. В частности ВЫРОВНЕННАЯ НЕЧЁТНАЯ θ=0 ячейка (s=t в нечёт)
имеет несингулярные представители.

**Аналитически** (census_notes.md, гомотопия): осевой зажим фиксирует n_end∈S²
(точку), слой свободен; любые осевые концы соединимы гладким путём на S² + нужное
число 2π-твистов слоя (гладкие пути) ⇒ любой ℤ₂-класс достижим при любых концах.
Выровненная нечётная θ=0: постоянный путь на S² + один 2π-твист слоя = гладкий
2π-твист-солитон. Нестрогие шаги честно помечены (Н1 бассейны — проверено численно;
Н2 сектор мягок при осевых зажимах; Н3 существование ≠ глобальный минимум).

**Вердикт D-H3a: NOT-CONFIRMED.** Выровненные ветви в нечёт θ=0 ИМЕЮТ несингулярные
представители. Сработал Kill калибровки («EXISTS во всех ячейках → запрет не
конструктивен → уходит в меру»). Геометрической пустоты носителя НЕТ; подавление
выровненной ветви (синглет фазы C) — энергетическое/энтропийное, не геометрическое.
**NOT-FOUND нигде не достигнут ⇒ вопрос о запрете не решается статикой.**

## 3. D1-B. Кандидаты считывания и слепота (`readout.py`)

Кандидаты (пара знаков левый/правый, окно k): AX (sign n_end·a, SO(3)-эталон);
A (знак w-компоненты считывающей нормали в R⁴); C (знак w ближайшего интерьерного
узла); Bloc_k (parity лифта окна k, k∈{2,4,8}).

**Тест слепоты** (пары u,u' с ТОЖДЕСТВЕННЫМ осевым полем n_i, но ПРОТИВОПОЛОЖНЫМ
классом лифта; u'=R(ψ_i,n_i)⊗u, ψ:0→2π, max|Δn|=0): среднее agree_frac по 24 ячейкам
динамики — **AX=1.0, A=1.0, C=1.0, Bloc2=1.0, Bloc4=1.0, Bloc8=1.0**. ВСЕ кандидаты
слепы к лифт-твину ⇒ **SO(3)-факторизуемы** (не различают ℤ₂-класс).

**Ключевой факт:** lift_twin переворачивает parity (+1→−1) при ТОЖДЕСТВЕННОМ n_i.
Переворот живёт в ГРАНИЧНОМ знаке лифта (u_end→−u_end), а локальные произведения
sign(g_i.w) СОХРАНЯЮТСЯ. ⇒ ℤ₂-класс — ГЛОБАЛЬНОЕ (граничное) свойство, локально не
считываемое физическим датумом. (Гипотетическое считывание u_end относительно
ФИКСИРОВАННОГО лаб-лифта различило бы классы, но лаб-лифт нефизичен — аппаратура
видит лишь SO(3)-ориентацию.)

**Тест локальности** Bloc(k): доля совпадений Bloc(k) с Bloc(8) = {k2:1.0, k4:1.0,
k8:1.0} ⇒ Bloc СТАБИЛЕН при k≤8 — ЛОКАЛЕН (не BAKED-IN).

приготовление (осевой вектор n_end равномерен на S² ⇒ бассейн решает ветвь). Все
цифры — СЫРЫЕ (s,t) по AX. `results/dynamics_full.json`.

## 5. Соответствие PREREG (включая сработавшие kill'ы)

**D-H3a (перепись).** Предсказание (пустота выровненной нечётной ветви) — **NOT-CONFIRMED.**
Все 16 ячеек EXISTS (численно + гомотопия); выровненная нечёт θ=0 имеет несингулярные
представители (2π-твист слоя). **Сработал Kill калибровки:** «EXISTS во всех ячейках →
запрет не конструктивный → уходит в меру». Геометрической пустоты носителя НЕТ.

**D-H3b (динамика).** Предсказание (частота запрещённой ветви = 0) — **НЕ выполнено:**
частота выровненной ветви в нечёт θ=0 ≈ 0.67 ≠ 0, ПЛОСКАЯ по T (нет exp(−c/T)).
- Kill-1 «энтропийный рецидив» (частота>0, убывает с T как exp(−c/T)): частично — есть
  лёгкое понижение odd vs even, но БЕЗ T-тренда ⇒ это НЕ термическая активация, а
  бассейн-геометрия. Строго «рецидив exp(−c/T)» НЕ подтверждён.
- Kill-2 «дырявый лифт-2» (нули за счёт роста отбраковок): **НЕ сработал** —
  rejected_singular/шаг ≈ 0 во всех ячейках; нулей нет, фильтр не работает.
- Итог D-H3b: запрета нет; выровненная ветвь населена ГЕНУИННО (не фильтром).

**D-H3c (считывание).** Двойственный результат — важно для интерпретации (с архитектором):
- По БУКВЕ prereg: A, C, Bloc локальны (Bloc стабилен k≤8) И отличаются от AX во всех
  ячейках ⇒ формально **D-H3c CONFIRMED** (локальный срезовый датум ≠ AX).
- По ПРЯМОМУ тесту слепоты: ВСЕ кандидаты (вкл. A/C/Bloc) слепы к лифт-твину ⇒
  SO(3)-факторизуемы ⇒ **ℤ₂-топологию НЕ читают**. Различие с AX — ГЕОМЕТРИЧЕСКОЕ
  (знак w-выпучивания, структура бассейнов), не топологическое.
- Согласование: срезовые датумы СУЩЕСТВУЮТ и несут информацию, ОТЛИЧНУЮ от осевой AX,
  но НИ ОДИН не различает два лифт-класса. Линия «считывание видит ТОПОЛОГИЮ» —
  ЗАКРЫТА (эффективный «слепой срез» в топологическом смысле), хотя буквальный
  критерий D-H3c (≠AX) выполнен. Носитель ℤ₂ — глобальный (граничный знак лифта),
  локально нечитаемый физическим датумом.

**Общий итог D1.** Все ТРИ калиброванные «вероятные» стены проявились: перепись не даёт
конструктивного запрета (D-H3a Kill); динамика населяет «запрещённую» ветвь генуинно
(D-H3b — запрета нет, фильтр не при чём); срез топологически слеп (D-H3c — топология
не считывается локально, хотя геометрические датумы ≠ AX). Сильный результат
(конструктивный геометрический запрет) НЕ получен — как и предсказывала калибровка.
Интерпретация двойственности D-H3c (буквальный ≠AX vs прямая слепота) — за архитектором.
