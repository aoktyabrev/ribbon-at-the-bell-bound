# C4-GHZ / G-T addendum 1 (архитектор): γ-вердикт по ДЫРА-N1. Коммит ДО β2.
γ-вердикт по ДЫРА-N1 (архитектор): рекомендация «M₃=4 ⇔ глобальная структура»
ОТКЛОНЕНА. Карандаш-конструкция: упорядочена только пара (1,2); s₁=ε(λ);
s₂=ε·σ(a₁,a₂), σ=−1 ⇔ (Y,Y); s₃=x₃(λ) при a₃=X, y₃(λ) при Y (детерминированные
±1). Тогда u−z=2, v+w=−2 ⇒ M₃=4; маргиналы ½; NS цел (ε униформна).
Биконди v2 (T3″(N)): «Мермин выше классического бонда достижим в M ⇔ M допускает
≥1 упорядоченную space-like пару (разделяемую, статистически невидимую)»;
глобальный порядок достаточен, НЕ необходим; промежуточных порогов по числу
упорядоченных пар НЕТ (одна пара уже даёт алгебраический максимум 4).
Оговорка отдельной строкой: утверждение — о ДОСТИЖИМОМ ЗНАЧЕНИИ Мермина, не о
воспроизведении полной GHZ-статистики (нулевые попарные при одной паре —
отдельный вопрос, регистрируется открытым).

β2-батарея (PR-пара + локальная третья, N=2e6): M₃=4±2σ; маргиналы ½ на всех
