## КАРАНДАШ КОНВЕНЦИИ (исполнитель, ОБЯЗАТЕЛЬНО до β3; ловушка нормировки названа)
Мермин-полином (A6-конвенция): M_N = Re(Π_j(a_j^X + i a_j^Y)) для НЕЧЁТНОГО N,
Im(Π…) для ЧЁТНОГО N (Ardehali) — **ловушка: Re на чётном N НЕ оптимален**.
Явно: M_N = Σ_{настройки} c(s)·⟨Π s_j⟩, c(s)=(−1)^{⌊#Y/2⌋} при #Y правильной
чётности (чёт для Re, нечёт для Im), иначе 0. Численно (перебор детерм.):
- **LHV-бонды:** N=2→2, N=3→**2**, N=4→**4**, N=5→4 (= 2^{⌊N/2⌋}; N=3 совпал с A6).
- **Квантовый (GHZ):** 2^{N−1} (N=3→4, N=4→8) при оптимальной фазе.
