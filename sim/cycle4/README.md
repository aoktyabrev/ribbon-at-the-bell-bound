### G-M (геометрия) — трёхконцевой закон связи
- Трёхконцевая лента / Y-развязка: закон связи P(s,t,r|a,b,c). Кандидаты-
  анзацы **из РАЗНЫХ идей** (урок цикла 3 «два+ семейства»), фиксация
  анзацев ДО батарей (garden-of-forking-paths защита, как C3-S A2).
- **МОНОГАМИЯ = ОБЯЗАТЕЛЬНЫЙ J-тест:** любой трёхконцевой анзац проходит
  проверку моногамии запутанности (CKW-неравенство / no-signaling
  моногамия корреляций) — провал моногамии = анзац отпадает (как no-signaling
