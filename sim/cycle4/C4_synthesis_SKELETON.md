## 1. theorem(N=3, value) — обобщение теоремы класса M
Ставка: M₃>2 достижимо ⇔ ≥1 упорядоченная space-like пара. Prereg edc5e6d.
Исход: **ТЕОРЕМА(N=3, значение)** — необходимость (T1‴ глоб. контрапозиция) +
достаточность (β2: одна пара → M₃=4, party3-детерм = значение-свидетель).
Оговорка: значение Мермина, НЕ полная GHZ-статистика. Коммиты: β2 7b8c371,
γ addendum2 840602b.

## 2. Трипартитное зеркало (шов + пересечение)
Ставка: внутр. геометрия даёт GHZ-подпись? Prereg 7bfca96/3298e54.
- **Шов (GM-F2):** чётность ⊂λ = LHV ⇒ GHZ-ПОДПИСЬ (попарные 0, тройной ±1) при
  M₃=2 (бонд) — «трипартитный шов», подпись без амплитуды. Предсказан 0.45.
  Коммит 99205c3.
- **Пересечение (GM-F2j):** чётность ⊂λ + ОДНА упорядоченная пара ⇒ M₃=4 + полная
  **GHZ-подпись при Мермин-настройках** (маргиналы ½, попарные 0). Развилка (i).
  Коммит ed8e950. СКОУП (после kinship): подпись при Мермин-настройках, НЕ
  genuine-N полная статистика (Свеличны S₃=4=biseparable, ca6d220).

## 3. Бинарный ярус размера блока (карандаш + Свеличны)
Ставка: ярус Мермина = функция структуры предшествования. Prereg v2 449b4ca.
Исход: под СТАНДАРТНЫМ Мермин-Клышко (после kinship-ретракции лестницы-k) —
ярус БИНАРЕН: любой локальный партнёр кэпит MK на 2.000; genuine-N (полный блок
m=N) достигает алгебр. 4.000 > квант 2.828 (класс-M блок = NS-бокс). Свеличны
GM-F2j S₃=4=biseparable подтверждает: пара покупает Мермина, НЕ Свеличны;
genuine-N требует блока size N. Коммиты: карандаш C4GT2_pencil-block, m=4 cb76cbd,
Свеличны ca6d220.
