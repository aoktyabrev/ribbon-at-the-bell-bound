## Анзац GM-F2 (уровень закона, зафиксирован до J)

## КАРАНДАШ GM-F2 (ДО J, зарегистрированные ожидания)
- **Попарные:** ⟨sᵢsⱼ|aᵢaⱼ⟩ = ⟨σᵢσⱼ⟩ = ⟨σ_k⟩ = 0 (чётность ⇒ σᵢσⱼ=σ_k, σ_k
  униформна) ⇒ **НУЛЕВЫЕ попарные ∀ настроек** — GHZ-подпись есть.
- **Тройной:** ⟨s₁s₂s₃⟩ = ⟨σ₁σ₂σ₃⟩ = **+1 ∀ настроек** (сохранённая чётность) ⇒
  |тройной|=1 (максимален) — GHZ-подпись есть.
- **Мермин:** M₃ = ⟨XXX⟩−⟨XYY⟩−⟨YXY⟩−⟨YYX⟩ = 1−1−1−1 = **−2**, |M₃|=**2** =
  LHV-бонд (НЕ 4). Чётность ⊂λ = LHV ⇒ Мермин не превышает бонд (G-T T1‴).
- **Моногамия (NS):** попарные=0 ⇒ моногамийный бонд тривиально выполнен.
- **ОЖИДАНИЕ (честное, A4):** GHZ-ПОДПИСЬ (0 попарные, ±1 тройной) ПРИ M₃=2 ⇒
  **стена «трипартитный шов»** (подпись без амплитуды). Прогноз 0.45.
