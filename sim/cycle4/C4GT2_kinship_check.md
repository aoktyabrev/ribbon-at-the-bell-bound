# C4-GHZ / G-T2 — СВЕРКА РОДНИ + РЕТРАКЦИЯ артефакта лестницы.
# Итог: климбинг-лестница (N=4 бьётся при k=2) — АРТЕФАКТ нестандартного оператора.
# Стандартный Мермин-Клышко (MK) её ОПРОВЕРГАЕТ. Батареи G-T2 не стартуют (ретаргет).

## Что случилось (сверка карандаша с опубликованными границами)
Мой G-T2 карандаш (C4GT2_pencil.py) использовал **нестандартный product-form**
оператор Мермина (Re/Im от Π(aˣ+iaʸ)) с коэффициент-структурой, у которой
отношение квант/LHV = 2 (не экспоненциальное). Опубликованная иерархия
([Sv87], [SS02], [CGPRS02]) использует **СТАНДАРТНЫЙ рекурсивный MK** (Belinskii-
Klyshko), у которого квант/LHV = 2^{(N−1)/2} (экспоненциальное).
⇒ мои «ярусы лестницы» и опубликованные границы — В РАЗНЫХ ОПЕРАТОРАХ.

## Пересчёт под СТАНДАРТНЫМ MK (нормир. LHV=1, квант=2^{(N−1)/2}):
Перебор ОПТИМАЛЬНЫХ NS-корреляций пар (не только PR):
- N=3, k=1: |M₃| = **2.000 = квант** (одна упоряд. пара достигает генуинного
  трипартитного кванта; бьёт LHV=1). ✓ (совпадает с теоремой N=3 value).
- N=4, k=1: |M₄| = **2.000** (бьёт LHV=1, но НЕ квант 2.828).
- **N=4, k=2: |M₄| = 2.000 — ТО ЖЕ, что k=1.** Вторая пара НЕ добавляет.
- N=5, k=2: |M₅| = 2.000 (< квант 4).
⇒ **ЛЕСТНИЦА НЕ КЛИМБИТ:** одна упоряд. пара даёт |M|=2 (потолок 2-producible),
больше непересекающихся пар НЕ поднимают под стандартным MK.

## РЕТРАКЦИЯ (реестр самокоррекции):
Клейм «N=4 бьётся при k=2, M₄=8=квант» (C4GT2_pencil, C4GT2_prereg_DRAFT,
cycle4/README, RELEASES) — **АРТЕФАКТ product-form оператора** (LHV=4 у него —
не стандартный MK). Под стандартным MK N=4 k=2 = 2.000, не 8. Приор «лестница
монотонна по k» 0.7 — ПРОИГРАН (k=2=k=1). Гипотеза климбинг-лестницы ОПРОВЕРГНУТА.

## Ответ на вопрос архитектора (Q1: совпадает ли ярус k-пар с m-сепарабельностью):
**ЧАСТИЧНО, НЕ как климбинг.** Соответствие с опубликованной иерархией:
- одна упоряд. пара + локальные = «2-producible» (nonlocality depth 2)
  корреляция; под MK она достигает |M|=2;
- при N=3 это = квант (генуинная трипартитность достижима одной парой);
- при N≥4 это НИЖЕ кванта 2^{(N−1)/2} — генуинная-N НЕ достижима
  непересекающимися парами — **ПОТОЛОК**, совпадающий с духом Свеличны/depth
  (2-producible < genuine-N для N≥4).
- Но КЛИМБИНГ (k=2 выше k=1) — НЕТ (мой артефакт). Ожидание 0.7 (совпадает как
  лестница) — НЕ взято; совпадает как ПОТОЛОК, не как климб.
Источники (сверено web 2026-07-22, вторичные обзоры — пометка): [SS02] Seevinck-
Svetlichny PRL 89, 060401 (2002); [CGPRS02] Collins et al. PRL 88, 170405 (2002);
[Sv87] Svetlichny PRD 35, 3066 (1987); обзор arXiv:2406.16202 (MK/Svetlichny
границы, нормировки). Svetlichny-граница по обзору = 2^{N−1} (m-сепарабельность,
un-norm), квант 2^{N−1}√2 — БИНАРНАЯ (genuine-N vs нет), не тонкий k-ярус.
