# C4-GHZ / G-T2 (в) — T2(N)-невидимость: адверсариальный проход. Статус: conjecture до γ.
# CRN-батарея: d612214 (4-корр точно тождествен ∀ порядков). Ниже — разбор кандидата.

## Кандидат (полный текст)
**T2(N):** для любого механизма класса M с precedence-структурой ≺ (частичный
порядок на N событиях): (a) все ДОПУСТИМЫЕ порядки разрешения (линейные
расширения ≺) дают ТОЖДЕСТВЕННУЮ совместную статистику P(s|a) (цепное правило);
(b) никакой функционал операциональной P-статистики не восстанавливает
фактический порядок (линейное расширение).

## Разбор + контрпримеры
### (a) — линейно-расширенная инвариантность (цепное правило)
Для ФИКСИРОВАННОГО совместного P любое линейное расширение ≺ реконструирует P
тождественно (цепное правило P=Π P(next|prev), порядок факторизации не меняет
произведение). Батарея d612214: N=4, 4 порядка, 4-корр разброс 0.000000. ✓ АНАЛИТИЧНО.

### Контрпример-попытка 1: СМЕШАННЫЕ порядки
Разные линейные расширения одного ≺ (напр. (1,2,3,4) vs (2,1,4,3)) — оба
топологически валидны для того же частичного порядка ⇒ то же P (цепное правило).
НЕ контрпример (это ровно (a)). Если «смешанный» = линейные расширения РАЗНЫХ ≺ —
см. ДЫРА-T2N-2.

### Контрпример-попытка 2: ЧАСТИЧНЫЕ транскрипты
R_X видит ЧАСТЬ данных e_Y без e_Y≺e_X. Разбор: (i) нарушает M3 (слепа к
не-предшествующим) ⇒ вне класса M; ЛИБО (ii) «видит» ⇒ e_Y фактически
предшествует ⇒ это упорядоченность (часть ≺), тогда (a) применима. Чисто —
классифицировано (как в L2c N=2).

### Контрпример-попытка 3: w-КОНДИЦИОНИРОВАНИЕ
Отклик упоряд. пары w зависит от настроек. Утекает ли порядок через P-функционал?
w влияет на КОРРЕЛЯЦИИ, не на маргиналы (NS, GM-F2j: маргинал party2 от a₁ = 0).
which-order — не P-функционал (операц. доступ = P). ⇒ не утечка. Чисто.

## ДЫРЫ (для γ):
- **ДЫРА-T2N-1 (оговорка, не брешь):** «никакой функционал P» — ОТНОСИТЕЛЬНО
  операциональной P-статистики. Наблюдаемые вне P (which-first-таймстамп,
  пост-транскрипт) не покрыты (M2 фиксирует наблюдаемое = P). Как L2c T2 — оговорка.
- **ДЫРА-T2N-2 (реальное уточнение):** «фактический порядок» = какое ЛИНЕЙНОЕ
  РАСШИРЕНИЕ (невидимо ✓), НЕ существование ≺. Существование precedence-структуры
  РАСКРЫВАЕТСЯ сверх-классическим значением (M>бонд / S>2 требует ≺). ⇒ клейм
  переписать: «невидимо КОНКРЕТНОЕ линейное расширение; существование ≺ — нет
  (оно и есть источник M>бонд)». Без этого (b) переусилен.
- **ДЫРА-T2N-3 (открытая):** инвариантность доказана для линейных расширений
  ОДНОГО ≺. Сравнение РАЗНЫХ ≺ (разные частичные порядки) может дать разное P
  (L1 N=2: collapse-модель порядко-зависима на joint). ⇒ (a) держится ВНУТРИ
  фикс. ≺; межструктурная инвариантность — НЕ клейм (и не должна быть). Назвать явно.

## ВЕРДИКТ ПРОХОДА: НЕ ЧИСТ (ДЫРА-T2N-2 уточнение + ДЫРА-T2N-3 область).
Рекомендация: клейм T2(N) переформулировать — (a) линейно-расширенная
инвариантность ВНУТРИ ≺ (чисто, батарея+аналитика); (b) невидимость КОНКРЕТНОГО
расширения (не существования ≺). Статус: **conjecture** до правки + γ.
Прогноз «≥1 дыра» — сработал (T2N-2/T2N-3). β (CRN) есть; α (правка) + γ нужны.
