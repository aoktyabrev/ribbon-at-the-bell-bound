# Цикл 2 — гипотезы архитектора. Статус: этап 0, черновики.
Ставки: C2-F (факторизация релаксации: flip-rate F при совпадающей λ,
маргинальный Δ); C2-TM (замкнутая форма A(k_f), held-out предсказание);
C2-ISO-DYN (динамическая изотропизация: зажимы=настройки в релаксации,
Haar-ориентация) — условна на исход C2-F; C2-L (карандаш: лемма
факторизации → невозможность μ(B)=|s·a−t·b|²/8 в классе аддитивных
зажимов). Порядок: F → TM → ISO-DYN. Методология цикла 1 обязательна:
prereg-коммит ДО сырья, сырьё→коммит→анализ, зеркала, DEGENERATE без
отсечения, изотропная CHSH-формула запрещена, S>2 = стоп-аудит.
Снято: R-neq (аттрактор измерен в DS3 — треугольник; линия поглощена
задачей μ(B)); флуктуационно-теоремная трактовка трилеммы (плато
атермально). Проблема E закрыта гейтом (нет фибро-ломающего датума).
