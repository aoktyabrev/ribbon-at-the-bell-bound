# C2-J addendum 3: dt-арбитраж (мини-prereg, коммит до сырья).
Тест сходимости бассейнов ЯВНОЙ схемой фазы D (метрика канона; НЕ adaptive,
НЕ прекондиционер). Ячейки k_c×{64,256}, N=32, M=256, b=−a (a=ê, b=−ê,
геометрия R1-выравнивания, A→1 при выровненных концах), T=0, общий seed/λ.

## Правило устойчивости (вычислено, записано до прогона)
Явный Эйлер: lr·λ_max(Hess) < 2. Клин зажима −k_c(n·a)²: Hess ~ 2·k_c ⇒
lr < 1/k_c. При k_c×mult (k_c=2·mult): lr < 1/(2·mult).
  k_c×64 (k_c=128): lr < 7.8e-3;  k_c×256 (k_c=512): lr < 1.95e-3.
dt0=5e-3 УСТОЙЧИВ при ×64, НЕустойчив при ×256 ⇒ для ×256 старт с dt0/4.

## dt-лестница: {dt0, dt0/4, dt0/16, dt0/64} (+dt0/256 для ×256).
Шаги ∝ 1/dt (равное физ.время потока: dt·steps = const = 20).

## Наблюдаемые: flip-доля s между соседними dt (λ-спаренная); A=|⟨s·t⟩|;
C_ee=⟨n_A·n_B⟩; распределение E_final (разброс vs k_c×1 — стекло-диагностика).

## Ветвление (регистрируется до данных)
(I) СХОДИМОСТЬ: flip(dt→dt/4) монотонно ↓ и последняя пара ≤ бином.пола ⇒
    бассейны определены. (Ia) A≥0.6 ⇒ совместный режим при квенче реален,
    лестница C2-J возобновляется с dt-правилом; стекло-(а) проиграна.
    (Ib) A<0.3 ⇒ квенч-стекло, декорреляция реальна, совместный кинетически
    заперт ⇒ go задача 3. Промежуток 0.3–0.6 — серая, стоп/архитектор.
(II) НЕСХОДИМОСТЬ: flip не убывает ⇒ РЕЗУЛЬТАТ «детерминированные исходы не
    определены в сильно-цепном режиме» (детерминизм умирает до границы Белла)
    ⇒ go задача 3 (T>0 делает исходы распределительными и корректными).
