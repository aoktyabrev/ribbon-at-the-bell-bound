# Черновик правки канона по итогам C2-ISO-DYN (утверждение — Артем).
Уточнение трилеммы: утверждение цикла 1 «честная изотропизация ⇒
треугольная LHV» — свойство ПРОТОКОЛА (пост-хок поворот замороженного
источника, DS3), не онтологии. Динамическая изотропизация (зажимы в
релаксации, Haar-ориентация источника, C2-ISO raw 3fd3e3d) даёт
изотропную косинусоподобную E(θ) при подавленной амплитуде
(ρ=0.143/0.336 при k_f×{1,4}), S=0.357/0.854 ≤ 2, факторизация
подтверждена (C2-F). Форма корреляции — свойство эффективного
локального правила отклика, реализуемого динамикой (битва правил,
канон §2.3): пост-хок знаковое считывание ⇒ треугольник; гладкий
релаксационный отклик ⇒ косинусоподобная (чистый косинус: фит семейства
хорды даёт p̂≈2.5±0.63/2.06±0.18, AICc предпочитает косинус — 81cbcf9;
гипотеза p<2 отвергнута).
Трилемма переформулируется: форма и изотропия СОВМЕСТИМЫ; стеной
остаётся амплитуда (столбы косинусных LHV применимы: ρ ≤ 1/K_G).
Формулировки статьи цикла 1 (§6.4 — точна для своего протокола;
abstract/§8.1 — избыточно общи) НЕ редактируются задним числом;
уточнение — материал статьи цикла 2.
