# C2-F addendum 1 (после F0, до F0b и основной кампании).
1. Разводка веток при F≡0: ветка (ii) «плато» означает ненулевой пол
   (насыщение F артефактом разрывности); F≡0 во всех точках лестницы —
   ветка (i) проход по определению «0 событий». Кодифицирует
   интерпретацию отчёта F0 (анализ d06f258).
2. Позитив-контроль F0b (гейт инструмента, N=32, M=1200, T=0):
   F_s при повороте СВОЕГО зажима a→a' на π/2 (b фикс.) и F_t при
   повороте b→b' на π/2 (если не извлечён из сырья 7f5d176).
   Прогноз: оба ≫ 0 (архитектор: ≳0.3). KILL F0b: любой из двух = 0
   ⇒ харнес не видит флипы ⇒ стоп-аудит; результаты F0 объявляются
   неинформативными до исправления.
3. Ансамбль-диагностика: маргиналы P(s=+), P(t=+) — обязательные
   колонки всех отчётов C2-F; стратификация F по квартилям начального
   полярного угла граничного репера (arccos(n_A·ê) на prep) —
   вторичная наблюдаемая, регистрируется до основной кампании.
4. Перевес основной кампании: первичная ячейка k_f×1 (полная сетка
   Δγ∈{π/8,π/4,3π/8,π/2}, N∈{32,96}, M=4800, T=0 первично, T=0.05
   вторично); k_f×4 — подтверждающая (Δγ∈{π/4,π/2}, N=32, M=4800).
   Kill-критерии H-F1/H-F2/H-F3 без изменений, применяются к
   первичной ячейке.
5. Диагностическая строка (НЕ kill): A(k_f×1) основной кампании
   сверяется с замороженным 0.418±0.026 (3σ_comb) — след флага V0.
