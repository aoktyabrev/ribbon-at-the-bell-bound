# C2-R prereg (релиз архитектора 2026-07-16; вето Артема пассивно). Граница режима κ*.
Мотив (C2-L): Факт 1 — κ→∞ даёт PR-box (CHSH=4); Факт 2 — измеренный режим
(κ_eff≈0.065, k_f×{1,4}) факторизуется (F_s<6.3·10⁻⁴). Между ними — граница
κ*(N,k_c). C2-R ищет излом, повышая k_f (⇒ κ_eff↑), и меряет прямую CHSH за ним.

## Стадия R0 — стабильность интегратора при больших k_f (ДО науки)
При больших k_f |∇E_twist| ∝ k_f ⇒ ланжевен-шаг lr·∇E растёт ⇒ неустойчивость
scan-carry (как fp-грабли цикла 1). Правило lr (предложение, на утверждение):
  lr = lr0 · min(1, 4/k_f_mult),  lr0=5e-3.
(k_f×8 ⇒ lr0/2; ×32 ⇒ lr0/8; ×128 ⇒ lr0/32; ×512 ⇒ lr0/128 — эффективный
шаг lr·k_f ограничен сверху.) Смоук M≤8: ТОЛЬКО rej_per_step и конечность
выходов (нет NaN/Inf, |u|=1), БЕЗ снятия наблюдаемых. KILL R0: rej_per_step
не стабилизируется (>0.01) или非конечности при выбранном lr ⇒ правило lr
пересмотреть до науки; при неудаче — неявный/адаптивный интегратор (решение
архитектора).

## Основная кампания (после R0-прохода)
F_s(k_f) при k_f×{8, 32, 128, 512}, N∈{8, 32} (κ-скейлинг длины), M=4800, T=0.
Протокол C2-F (сопряжённые прогоны, общий λ, a=ê фикс, b→b'(Δγ), считывание ê;
DEGENERATE колонкой, sign(0)→+1, зеркала). Первичный Δγ=π/2 (максимальный
рычаг); при F_s>0 в ячейке — доснять полную сетку Δγ∈{π/8,π/4,3π/8,π/2} И
прямые 4 CHSH-коррелятора ТАМ ЖЕ (стандартная геометрия a∈{0,π/2}, b∈{π/4,3π/4}).

## Гипотезы (прогнозы архитектора — в журнал до данных)
H-R1 (излом существует): F_s > 3σ_F при некотором k_f ≤ 512, N=32.
  KILL: F_s < 3σ_F на всей лестнице k_f до 512 (N=32) ⇒ факторизация
  устойчива до κ_eff(×512); граница за пределом окна — режим шире, чем
  ожидалось, PR-предел (Факт 1) достигается только в строгом κ→∞.
H-R2 (κ-скейлинг): излом при МЕНЬШЕМ k_f для N=8, чем для N=32 (короче лента ⇒
  выше эффективная жёсткость на длину). KILL: одинаковый порог по k_f для
  обоих N (в пределах шага лестницы) ⇒ κ* не масштабируется длиной как ожидалось.
H-R3 (S_direct): при F_s>0 прямая CHSH растёт от 2ρ (LHV) вверх.
  ЛЮБОЕ S>2 — СТОП-АУДИТ ДО интерпретации (Пирл/симметрия/no-signaling
  маргиналов проверяются ПЕРВЫМИ, стандарт цикла 1). Интерпретация «совместный
  режим / выход за LHV» ЛЕГАЛЬНА только после: (i) аудит пройден, (ii) F_s>0 в
  той же ячейке, (iii) маргиналы P(s|a), P(t|b) no-signaling (независимы от
  удалённой настройки в пределах σ). Без всех трёх — S>2 = артефакт до доказательства.
H-R4a (форма-траектория, главный выход). На каждом k_f: полная θ-сетка
(9 точек) + прямые 4 CHSH-коррелятора. Фит E(θ) ТРЕМЯ моделями:
  (1) чистый косинус −ρ·cosθ;
  (2) СЕМЬЯ ОТКЛИКА (нулевая модель факторизованного режима):
      E_β(θ) = −∫ g_β(λ·a) g_β(λ·b) dλ/4π, g_β(c)=tanh(βc)/tanh(β),
      параметры (ρ_или_норм, β); численная квадратура в анализе;
      β→0 ⇒ косинус/3, β→∞ ⇒ треугольник;
  (3) семья хорды (ρ, p) — модель совместного режима.
AICc-таблица по всем k_f. Регистрируемые ожидания: в ячейках с F_s≈0
выигрывает (1) или (2) с β̂, растущим по k_f; выигрыш (3) или провал
всех трёх в факторизованном режиме — сюрприз-ветка, стоп до разбора.
Инструментовка ОБЯЗАТЕЛЬНАЯ: per-replica хранится R (кватернион
поворота prep) — для функции отклика P(s=+ | (R·ê)·a) как прямой
диагностики механизма (бины по 0.1 в косинусе угла). Чек-лист
prereg↔JSON до запуска (правило исполнителя, теперь протокол).
H-R4b (СТОЛБЫ): пока F_s < 3σ_F, требуется ρ_dyn ≤ 1/K_G(3) ≈ 0.66. Нарушение
  (ρ_dyn > 0.66 при факторизующейся карте) = ОБЯЗАТЕЛЬНЫЙ стоп-аудит:
  конфликт с теоремой Гротендика ⇒ по построению баг пайплайна, не физика.
H-R5 (характер излома; приоры релиза): КРОМКА (резкая граница, механизм семьи
  отклика) 0.35; РАННИЙ СЛОМ (постепенный, до 512) 0.40; СЕРАЯ 0.25.
H-R6 (механизм; функция отклика): P(s=+|c), c=(R·ê)·a — линейна (1+κc)/2 в
  факторизованном режиме ⇒ рычаг ρ = κ/3. KILL-сюрприз: знаковая ступень
  (не линейна) при F_s≈0 ⇒ отклик не гладкий, пересмотр нулевой модели (2).
РЕЛЬС (R0-стабильность): лестница едет ТОЛЬКО пока R0-гейт держится на каждом
  k_f (rej_per_step < 1e-3 И якорь A(k_f×4) в 3σ_comb на входе). Срыв рельса =
  немедленный стоп-отчёт (не в финале), пересмотр lr-правила до продолжения.
[Исполнитель: H-R5/H-R6/рельс формализованы из приоров и тем релиза (задачи 4-5);
 архитектор верифицирует пассивно. H-R1..H-R4 — из ранних этапов.]

- хорда-фит закрывает S-дефицит — приор 0.55. [ИСПОЛНИТЕЛЬ: НЕ закрыл; дефицит
  = ярд-мерка (ρ_tri vs ρ_cos) + ~1σ анизотропия, не p<2.]
- столбовый потолок 0.66 не нарушается в факторизующем режиме — приор 0.9.
- излом F_s к k_f ≤ 512 (N=32) — приор 0.6.

## Журнал (обновления при prereg-коммите, с причинами)
1. Дуэль H-I0: вердикт БЕЗ изменений; вторая линейка на ρ_cos-эстиматоре:
   архитектор 0.29±0.10 vs модель-free 0.336 vs ФАКТ ρ_cos=0.310±0.010 (×4).
   Правило: прогнозы называют эстиматор (ρ_tri vs ρ_cos не взаимозаменяемы).
2. Хорда: ОБА прогноза архитектора (p̂∈[1.4,1.8]/[1.6,1.9]; хорда закрывает
   S-дефицит) ПРОИГРАНЫ на сырье 81cbcf9 (p̂≈2.5/2.06, косинус, дефицит =
   ярд-мерка не форма).
3. Приоры H-R5: кромка 0.30→0.35 (механизм семьи отклика), ранний слом 0.40,
   серая 0.25. Остальные приоры релиза без изменений.
4. Новые прогнозы: форма-фит выигрывает семьёй (1)/(2) в факторизованном
   режиме — 0.7; β̂ монотонен по k_f — 0.6; функция отклика ISO-DYN линейна
   (κ̂/3 = ρ_cos в 2σ) — 0.5.
5. Инструментовочный пробел ISO-DYN: per-replica R НЕ сохранён в raw 3fd3e3d
   (не дефект prereg — R не был в списке наблюдаемых ISO); диагностика
   P(s=+|c) из замороженного сырья НЕВОЗМОЖНА, закрыта требованием H-R4a/H-R6.

## Бюджет (оценка, fp64 GPU; ОБНОВЛЁН под H-R4а полную θ-сетку)
R0 смоук: секунды. Две компоненты на ячейку (k_f×N):
 (i) F_s(k_f): сопряжённые прогоны Δγ=π/2 (первичный) × зеркала(2) = ~3 прогона;
 (ii) E(θ) шейп H-R4а: полная θ-сетка 9 точек + 2 доп. CHSH-угла = ~11 прогонов
      (как ISO-DYN, ~8 мин/ k_f-ячейка при M=4800).
4 k_f × 2 N = 8 ячеек × ~14 прогонов = ~112 прогонов M=4800. При больших k_f
lr мал ⇒ то же число шагов (6000). Оценка: ~60–90 мин (θ-шейп доминирует над
F_s-лестницей). Сокращение: E(θ)-шейп только на N=32 (κ-скейлинг F_s на N=8
достаточно точкой излома) ⇒ ~40–50 мин. РЕШЕНИЕ по объёму θ-шейпа — за Артемом
(вопрос 2/3 открыты).
