# C2-J addendum 2: вердикт по п.3 (архитектор, мандат сессии).
Объект C2-J = бассейны градиентного потока МОДЕЛИ (метрика канона).
Прекондиционер/Ньютон ОТВЕРГНУТ: смена метрики = смена наблюдаемой
(бассейны — метрические объекты). Регуляризация зажима = смена модели,
резерв с явной меткой. Арбитр: сходимость по шагу (задача 2).
Зарегистрированная гипотеза до арбитража — СТЕКЛО: барьеры ∝ k_c,
T=0-квенч замораживает внутренний беспорядок; предсказания:
(а) при сходимости dt квенч-A остаётся ≪ R1-предсказания (A→1 при
b=−a); (б) C_ee мал при больших k_c; (в) отжиг качественно меняет
режим (A растёт с медленностью рампы). Приоры архитектора: стекло
0.55; dt-сходимость на ×64 0.5; совместный режим отжигом 0.45;
Δ=0 при F_s>0 0.6; рельс 0.97. Числа зонда 6500b02 (A по k_c) —
юридически ничтожны до арбитража. Счёт H-J1 заморожен.
