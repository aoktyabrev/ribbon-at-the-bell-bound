# C3-B prereg DRAFT (не утверждён). «Борн из причинности» внутренне —
# закрыть импорт партиально-запутанных ансамблей.

## Цель
Канон §3.2: no-signaling на партиально-запутанной рулёжке форсирует p=2,
НО ансамбли рулёжки импортированы из КМ (born2.py:20-24 «эксп. факт:
частично запутанные пары существуют»; {q,1−q} и {50/50 наклон} —
КМ state-steering). Закрыть импорт: асимметричный закон ленты
P(s,t|a,b,χ) должен ПОРОДИТЬ рулёжку ВНУТРЕННЕ, тогда no-signaling
даст p=2 без КМ-формул.

## Батарея свойств (ЯДРО; источник — born2/chord, НЕ КМ-формулы)
Закон P(s,t|a,b,χ) обязан удовлетворять, чтобы теорема замкнулась внутренне.
Каждое свойство — исполняемым assert'ом на numpy-батарее (упрочнение цикла 2).

(B1) РЕПИТАБЕЛЬНОСТЬ/собственные концы (born2 f(±1)∈{0,1}, chord.py:9-11
  вырожденные ветви): P(s=+|a,χ)→1 при n·a→+1, →0 при n·a→−1. Исходы ±1.
(B2) ЭКВАТОРИАЛЬНАЯ НЕсмещённость (born2 f(0)=1/2): при n·a=0 P(s=+)=1/2.
(B3) NO-SIGNALING/маргинал (born2 Батарея3 требование Δ=0, born2.py:35-39;
  chord.py:68): P(s|a,b,χ)=P(s|a,χ) НЕзависим от удалённой b; симметрично
  P(t|a,b,χ)=P(t|b,χ). ЦЕНТРАЛЬНАЯ связь.
(B4) СРЕДНЕ-ДОСТАТОЧНОСТЬ ⟺ АФФИННЫЙ отклик (born2 Батарея2, born2.py:8-14:
  два ансамбля с одним средним различимы ⟺ f нелинейна): ⟨отклик⟩ по
  ЛЮБОМУ ансамблю = отклик(⟨n·a⟩) ⇒ f аффинна в проекции c=n·a ⇒
  f=(1+c)/2 = Борн (p=2). Это внутренний форсер p=2.
(B5) ВНУТРЕННЯЯ РУЛЁЖКА (ЗАКРЫВАТЕЛЬ импорта): закон при фиксированном χ
  и разной удалённой настройке даёт ДВА разложения ближнего ансамбля с
  РАВНЫМ средним ⟨n·a⟩, но РАЗНЫМИ высшими моментами (рулёжная
  диверсность, что КМ даёт партиальной запутанностью). Порождается
  структурой χ ленты, НЕ импортом. Без B5 у B4 нет предмета.
(B6) ГРАНИЧНЫЕ по χ: χ=0 → произведение (E=0, рулёжки нет, разложения
  тривиальны); χ→χ_max → максимальная (анти)корреляция; диверсность B5
  существует при промежуточном χ; E(χ) монотонна при фикс θ.
(B7) ЦЕПНЫЕ/ПОРЯДОК (репитабельность под композицией): пост-измеренческое
  χ-обновление ассоциативно для последовательных измерений; повтор a →
  тот же s; совместный закон порядко-консистентен (χ несёт причинную
  асимметрию, но наблюдаемый no-signaling держится в любом порядке).

ТЕОРЕМА замыкается ВНУТРЕННЕ ⟺ лента даёт B1,B2,B5,B6,B7 (рулёжка из своей
χ-структуры) И тогда no-signaling B3+B4 форсирует аффинность = Борн (p=2).
Импорт #2 закрыт ИМЕННО B5.

## Семейства-кандидаты (≥2, РАЗНЫЕ геом. идеи; ≤2 своб. пар. + χ)
- **F1 взвешенная хорда:** P(s,t|a,b) ∝ |s·√w_a(χ)·a − t·√w_b(χ)·b|^p,
  w(χ) — асимметричные веса концов (χ смещает баланс). Обобщает chord.py
  (симметрия w_a=w_b=1); χ вводит асимметрию концов.
- **F2 смещённая мера скрутки источника:** плотность микрооси λ у источника
  = (n·λ)_+ · (1 + χ·(m·λ)) — смещение КС-меры (chord.py:43-52) вектором
  асимметрии m(χ). χ рулит форму разложения при фикс среднем.

## KILL-критерии (регистрируются)
- Провал ЛЮБОГО из B1,B2,B6,B7 семейством ⇒ семейство ОТПАДАЕТ.
- B5-провал (нет рулёжной диверсности внутренне) семейством ⇒ у B4 нет
  предмета ⇒ семейство НЕ закрывает импорт.
- Провал ВСЕХ зарегистрированных семейств по B5 ⇒ СТЕНА: рулёжка
  невыводима внутренне в классе ⇒ импорт неустраним ⇒ фиксируется
  ИСХОДОМ (не провалом); канон §3.2 остаётся честным «within ansatz».
- ТЕСТ ЕДИНСТВЕННОСТИ p=2: телеграф-скан Δ(p,χ) по (p,χ)-сетке
  (p∈{1,1.5,2,2.5,3,∞}, χ-сетка). Регистрируемое: Δ(p=2,χ)=0 ∀χ (в пределах
  MC-шума 1/√N) И Δ(p≠2,χ)>3σ при χ, дающем рулёжку. Провал (Δ(2,χ)>0
  где-то) = баг батареи ИЛИ p=2 не форсируется этим семейством.
