# C3-B addendum 2: коррекция гейта робастности (архитектор).
Диагноз: гейт кода (analytic_zero_set, tol=1e-9 на квадратуре)
отсутствует в prereg и валит тест на ошибке трапеции ~2.3e-5
(сходимость 1/ngrid → 0 продемонстрирована). Зарегистрированный
критерий (addendum1/A4): числовое нуль-множество = {2} ∧
|аналитика − численность| < 2σ. КОРРЕКЦИЯ: гейт кода приводится
к prereg; ДОПОЛНИТЕЛЬНОЕ УСИЛЕНИЕ: в точке p=2 аналитический Δ
вычисляется алгебраически (аффинность f₂ ⇒ зазор Йенсена ≡ 0
тождественно), не квадратурой; требование — точный ноль.
Вердикты «СТОП» прогона 19:39 признаются ложными срабатываниями
гейта, не физики. В стандарт программы: prereg↔gate — построчная
сверка assert'ов кода с текстом prereg до первого прогона;
сырьё коммитится до правок кода.
