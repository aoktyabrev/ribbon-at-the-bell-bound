# C3-L addendum 1 (архитектор): фактор-2 и канонизация joint-модели.
1. L3-мост: финдинг принят как структурный. Теорема расслаивается:
   ОТБОР ПРАВИЛА (no-signaling + D>0 ⇒ p=2) — слоенезависим,
   установлен; ЗНАЧЕНИЕ S(p=2) — свойство механизма разрешения:
   shared-λ ⇒ ≤2 (измерено √2 при χ=0.5), совместный хордовый ⇒ 2√2.
   «Tsirelson from no-signaling» = верхний обрез неаффинного хвоста
   в совместном слое (S∈(2√2,4] исключён); амплитуду no-signaling
   не поднимает. АМПЛИТУДНЫЙ ШОВ регистрируется центральным долгом:
   внутренняя модель с D>0 ∧ S(2)=2√2 — построение = замыкание
   реконструкции; невозможность = фундаментальность шва. Секция
   статьи 3 переписывается по этому расслоению (текст — архитектор).
2. L1: канонизация Q1 (симметричная факторизованная) как СВИДЕТЕЛЯ;
   лемма экзистенциальна: «статистика внутреннего семейства
   допускает порядко-инвариантную реализацию» — PASS по построению.
   Порядко-зависимость коллапсной реализации (0.0418 при инвариантных
   маргиналах) — улика L2c, не патология: механика несёт порядок,
   статистика — нет. Прогноз L1 0.7 — проигран архитектором
   (недоспецификация joint-модели), в журнал.
3. L2: арена переносится на СОВМЕСТНЫЙ закон хорды p=2 (E=−cosθ,
   S=2√2), где факторизованная реализация запрещена Беллом и вопрос
   фолиации принудителен. Внутреннее семейство в L2 — только как
   контраст «локальному фолиация не нужна».
