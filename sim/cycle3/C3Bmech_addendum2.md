# C3-B-mech addendum 2 (архитектор): протокол источника для M1/M2. Коммит ДО сырья.
КАНОН: **fast-freeze** — плато T_prep=0.5 при поле h=0.05 → БЫСТРЫЙ квенч T→0
С ПОЛЕМ → выключение h на замороженном ансамбле → измерит. динамика КАНОНИЧЕСКАЯ
(без поля). Обоснование: мера μ_χ обязана быть свойством ИСТОЧНИКА (замороженная
намагниченность), не действующей силы — field-on на измерении менял бы премису
теоремы отбора.
Калибровка стадии: кривая **χ_frozen(rate)** по сетке скоростей квенча
{300, 600, 1200, 2000 шагов}; ансамбль M1/M2 = **самая быстрая ступень** с
валидным фитом формы 1+χn_z (χ²/dof<2). KILL протокола: ни одна ступень не
держит χ_frozen>3σ с валидной формой ⇒ фронт A стопорится стеной «биас источника
не переживает заморозку» — фиксируется.
КОНТРОЛЬ ЧИСТОТЫ (обязателен): ансамбль h=0 через тот же fast-freeze — все
наблюдаемые M1/M2 на нём обязаны дать нуль-эффекты (D_mech<2σ, маргиналы ½);
ненуль = артефакт протокола, стоп.
Рифма C2-J: выживание биаса = стекольная заморозка, χ_frozen(rate) — та же
физика, что E_std(rate) отжига.

## Прогнозы (обновление, причина «протокол определён»):
M1 (D_mech>0 на механич. ансамблях) — 0.55; M2 (нуль-множество {2}) — 0.45.
