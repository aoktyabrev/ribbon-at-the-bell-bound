# C3-S аналитика ДО батарей (A1/A2/A4/A7). Карандашные выводы S(χ),
# χ_max^dop, знаковая конвенция. Коммит в prereg ДО J-батареи.
# Свободный подбор запрещён (A2); вариации анзаца — новым addendum «попытка №N».

## Знаковая конвенция (A7)
Целевой знак совместного слоя: **E(θ|χ_max) = −cosθ** (синглет, антикорреляция;
рифма с антикорр. ядром ленты цикла 1). Внутренние C3-B-ансамбли давали
+cosθ (стиринг-маргинальный слой) — ДРУГОЙ слой; здесь совместный (хордовый)
антисимметричен. Единая конвенция: joint E<0 при малых θ. Расхождение
знаков между слоями названо явно (источник тихих багов).

## S-F1 (взвешенная совместная хорда), анзац A2: α=√(1+χ), β=√(1−χ)
P_χ(s,t|a,b) ∝ |s·α a − t·β b|². Раскрытие (|a|=|b|=1):
  |sα a − tβ b|² = α² + β² − 2st·αβ·cosθ.
Σ_4 branches = 4(α²+β²) (член cosθ·Σst=0). ⇒
  **P(s,t|a,b) = [ (α²+β²) − 2st·αβ·cosθ ] / [4(α²+β²)]**.
Маргиналы: P(s|a)=½ ∀χ (Σ_t обнуляет cosθ-член) ⇒ J3 no-signaling ✓.
Корреляция:
  E(θ|χ) = −[2αβ/(α²+β²)]·cosθ = **−√(1−χ²)·cosθ**  (α²+β²=2, 2αβ=2√(1−χ²)).
CHSH:  **S_F1(χ) = 2√2·√(1−χ²)**.  [ЗАРЕГИСТРИРОВАННАЯ КРИВАЯ]
- χ=0: E=−cosθ, S=2√2 (синглет), но α=β ⇒ СИММЕТРИЯ (нет асимметрии).
- χ=1 (=χ_max^dop): β=0 ⇒ E=0, продукт (J1-конец).
- Фактор 2αβ/(α²+β²) ≤ 1 (AM-GM), =1 ⇔ χ=0. **ЛЮБАЯ асимметрия χ>0
  снижает S ниже 2√2.** Позитивность: √(1−χ²)|cosθ|≤1 ∀χ∈[0,1] ⇒ χ_max^dop=1.

## S-F2 (смещённый источник × совместное разрешение), анзац A2 — попытка №1
Источник λ~μ_χ ∝ 1+χλ_z (позитивность: 1−χ≥0 ⇒ **χ_max^dop=1**).
Пер-λ совместное разрешение = смесь хорда/продукт с λ-выравниванием:
  P_λ(s,t|a,b) = q(λ)·[хорда (1−st cosθ)/4] + (1−q(λ))·[продукт ¼],
  q(λ) = (1+λ_z)/2 ∈[0,1] (выравнивание с осью смещения).
Статистика = λ-среднее (μ_χ). ⟨q⟩ = (1+⟨λ_z⟩)/2, ⟨λ_z⟩_μχ = χ/3 ⇒ ⟨q⟩=(1+χ/3)/2.
  E(θ|χ) = −⟨q⟩·cosθ = −[(1+χ/3)/2]·cosθ.
  **S_F2(χ) = 2√2·(1+χ/3)/2 = √2·(1+χ/3)**.  [ЗАРЕГИСТРИРОВАННАЯ КРИВАЯ, попытка №1]
- χ=0: S=√2 (полусинглет); χ=1: S=√2·4/3 ≈ 1.886. **НИКОГДА не 2√2.**
- Маргиналы: P(s|a,λ)=½ (и хорда, и продукт симметричны) ⇒ J3 ✓.
- D_joint из q(λ)-смешивания (λ-диверсность), χ>0.

## СТРУКТУРНОЕ НАБЛЮДЕНИЕ (в prereg, до батарей — предсказание вердикта)
Оба анзаца разводят D>0 и S=2√2 в РАЗНЫЕ стороны:
- S-F1 достаёт 2√2 только при χ=0 (симметрия), но там нет источника
  асимметрии; при χ>0 (нужном для внутр. диверсности) S<2√2. Причём
  S-F1 — ПОСТУЛИРОВАННЫЙ совместный закон (нет скрытой непрерывной меры)
  ⇒ D_joint (3-й момент условных НЕПРЕРЫВНЫХ ансамблей) не определён/0 ⇒
  J4 (внутр. генерация) провалится: диверсность НЕ порождается мерой.
- S-F2 порождает D_joint>0 из μ_χ, НО это λ-СРЕДНЕЕ ЛОКАЛЬНЫХ откликов ⇒
  LHV ⇒ **S≤2 по Беллу** < 2√2. Достать 2√2 λ-средним локальных откликов
  нельзя (теорема Белла).
⇒ ПРЕДСКАЗАНИЕ: ни одно семейство не закрывает шов. Структурная причина =
БЕЛЛ: внутренняя генерация (скрытая мера + локальный отклик) = LHV = S≤2;
2√2 требует НЕЛОКАЛЬНОГО совместного разрешения (постулат закона ЛИБО
preferred-foliation joint-разрешение L2(i)). **Шов ≡ вопрос фолиации (L2c):
закрыть шов = требовать нелокальное совместное разрешение.** Это делает
C3-S улику СИЛЬНОЙ, но «невозможность»-теорема — только через A5 α/β/γ
(строгий Белл в названном классе + ревью). Числа кривых проверяет батарея.
