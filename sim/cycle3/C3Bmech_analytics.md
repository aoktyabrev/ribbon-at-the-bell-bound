# C3-B-mech — аналитика ДО батарей (стандарт A1). Вердикт до GPU.
# Вопрос: механический стиринг (μ_χ из упругой динамики) = F2-стиринг?
# Или стена «конусное срезание убивает разнообразие»?

## Постановка
μ_χ ∝ 1+χ n·ẑ (n — ось связи на S²); знаковое считывание s=sign(n·a).
Аналитика ГИВЕН μ_χ (полагаем источник поляризован; поляризацию проверяет
M0-батарея). Числ. контроль N=4·10⁶ рядом.

## (1) Условные меры P(n|s,a) — полусферные срезы
P(n|s=+,a) = μ_χ(n)·1{n·a>0} / P(s=+|a) — срез смещённой меры полусферой
{n·a>0}. Аналогично s=− → {n·a<0}. Конус = полусфера (знаковое считывание).

## (2) Условные средние m(s,a) и МЕАН-СОГЛАСОВАННЫЕ пары
m(s,a) = E[n | s·(n·a)>0] под μ_χ (вектор).
- **Зеркальная пара (a,+)↔(−a,−):** {n·a>0} = {n·(−a)<0} ⇒ ОДИН И ТОТ ЖЕ
  ансамбль ⇒ m совпадают ТОЖДЕСТВЕННО (вырожденная пара, для D бесполезна:
  нужны РАЗНЫЕ ансамбли при равных средних).
- **Рабочая пара a=ẑ vs a=x̂ (закон полной вероятности):** проекция усл.
  среднего на ẑ, взвешенная по полусферам:
  mom(a,1) = P₊(a)·(m₊·ẑ) + P₋(a)·(m₋·ẑ) = ⟨n_z⟩ ∀a (закон полн. ожидания).
  ⇒ **mom(ẑ,1) = mom(x̂,1) = ⟨n_z⟩ = χ/3 — РАВНЫЕ СРЕДНИЕ автоматически.**
  Числ.: |Δmean(ẑ,x̂)| = 0.00000 ∀χ (контроль). Континуум a при равном
  среднем ⇒ карта средних плоская по настройке (первый момент = χ/3 везде).

## (3) Третьи моменты при равных средних — D_mech(χ)
D_mech(χ) = |mom(ẑ,3) − mom(x̂,3)| (3-й момент проекции усл. среднего,
равные первые). Числ. контроль:
  χ:    0.0     0.25    0.5     0.75    1.0
  D_mech: 0.0003  0.0306  0.0570  0.0759  0.0824
**D_mech(χ) > 0 при χ>0** (растёт монотонно; совпадает с C3-B D(χ) 0.03→0.08).
⇒ Знаковое конусное срезание НЕ убивает разнообразие: полусферы {n_z>0} и
{n_x>0} имеют равный ⟨n_z⟩, но РАЗНЫЙ 3-й момент (концентрация против
размазки) ⇒ стиринг-диверсность есть.

## (4) Прогноз кривых (регистрируется до батарей)
- **Маргинал-биас:** P(s=+|a,χ) = ½ + (χ/4)·(a·ẑ) = ½ + χ a_z/4.
  Числ.: P(+|ẑ)=0.5625/0.625/0.6875/0.75 при χ=0.25/0.5/0.75/1.0 ✓;
  P(+|x̂)=½ ∀χ ✓. [M0-калибровка мерит именно этот биас vs поле h.]
- **E(θ|χ):** внутренний стиринг = суб-единичное усл. среднее (как L3
  ячейка2) ⇒ S(2)≤2 (при χ=0.5 → √2). Механизм наследует шов (не 2√2).

## ВЕРДИКТ АНАЛИТИКИ (до GPU)
**Фронт A ЖИВ.** D_mech(χ)>0 аналитически ⇒ стена «конусное срезание
убивает разнообразие» ОПРОВЕРГНУТА: механический стиринг = F2-стиринг
(та же структура полусферных срезов, тот же D). Провалившегося свойства
НЕТ на уровне меры.
**Make-or-break перенесён на M0 (эмпирика GPU):** порождает ли УПРУГАЯ
динамика меру μ_χ с χ>0? Т.е. поляризуется ли источник выравнивающим полем
h, ИЛИ релаксация смывает биас (стена M0 «источник не поляризуется»). Это
НЕ аналитический вопрос — упругая динамика цикла 2, GPU. Аналитика фронта A
исчерпана: премиса отбора (D>0) держится ГИВЕН поляризацию; вопрос —
поляризация.
