# Ландмарк канона: «точка 1/3-косинуса» (the 1/3-cosine point).
# Статус: канон-заметка (landmark). Три независимых выхода → одна точка.

## Определение
**Точка 1/3-косинуса** — корреляционная статистика E(θ) = (1/3)·cosθ
(|E|≤1/3, S = 2√2/3 ≈ 0.943 ≤ 2), максимальная изотропно-косинусная
статистика, доступная ЛОКАЛЬНОМУ ЛИНЕЙНОМУ считыванию скрытого вектора
λ на сфере. Аналитически: для λ равномерно (или μ_χ — второй момент не
меняется, χ-поправки нечётны ⇒ 0) E[λλᵀ]=(1/3)I ⇒
E(a,b)=E_λ[(2f₂(λ·a)−1)(2f₂(λ·b)−1)]=E_λ[(λ·a)(λ·b)]=aᵀ(⅓I)b=cosθ/3.
Это ПОТОЛОК shared-λ слоя (нижняя граница амплитудного шва до 2√2).

## Точка: аналитический якорь + два механизма (ISO-DYN брекетит)
**Аналитический якорь (точно 1/3):** правило-рычаг — локальное линейное
считывание f₂ скрытого вектора ⇒ E=cosθ/3 ТОЧНО (вывод выше). Плюс frame-local
E(0)=0.333.
1. **Правило-рычаг (аналитически, точно).** E=cosθ/3. battery_c3l_l2.py; **49e8c1b**.
2. **L2 frame-local space-like (механизм).** «Каждый первый» без совместного
   закона ⇒ факторизованный LHV: E(0)=0.333, E(90)=0, E(180)=−0.333, S=0.943.
   **49e8c1b** (C3L_L2.json).
3. **ISO-DYN (динамич. изотропизация, цикл 2) — БРЕКЕТИТ, не точно.** dynamical
   isotropization lands NEAR the point: ρ_cos=0.311±0.010 (cosine-fit, the
   form-matched estimator), ρ_model-free=0.336 (C2ISO_analysis.json, **a043f8f**)
   — bracketing 1/3 within ~2σ. Эстиматоры названы (правило программы). Сырьё
   3fd3e3d, prereg 77f734c.

## Значение для программы
Anchored analytically and reached by two mechanisms, with ISO-DYN bracketing it
⇒ это НЕ артефакт одной конструкции, а структурная граница локально-линейного слоя. Верхний
берег шва — 2√2 (совместный хордовый). Расстояние 1/3 → 2√2 = амплитудный
шов, центральная задача C3-S. «Стена цикла 2» (реляксация не достаёт
квантовых корреляций) и «шов реконструкции» (shared-λ не достаёт 2√2) —
одно и то же, с двух сторон.
