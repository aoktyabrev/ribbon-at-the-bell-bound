# Ландмарк канона: «точка 1/3-косинуса» (the 1/3-cosine point).
# Статус: канон-заметка (landmark). Три независимых выхода → одна точка.

## Определение
**Точка 1/3-косинуса** — корреляционная статистика E(θ) = (1/3)·cosθ
(|E|≤1/3, S = 2√2/3 ≈ 0.943 ≤ 2), максимальная изотропно-косинусная
статистика, доступная ЛОКАЛЬНОМУ ЛИНЕЙНОМУ считыванию скрытого вектора
λ на сфере. Аналитически: для λ равномерно (или μ_χ — второй момент не
меняется, χ-поправки нечётны ⇒ 0) E[λλᵀ]=(1/3)I ⇒
E(a,b)=E_λ[(2f₂(λ·a)−1)(2f₂(λ·b)−1)]=E_λ[(λ·a)(λ·b)]=aᵀ(⅓I)b=cosθ/3.
Это ПОТОЛОК shared-λ слоя (нижняя граница амплитудного шва до 2√2).

## Три независимых выхода на ОДНУ точку
1. **ISO-DYN (динамическая изотропизация, цикл 2).** ρ_measured=0.335
   (косинус, S≤2; третий угол трилеммы). Коммит **a043f8f**
   (cycle2-ISO analysis), сырьё 3fd3e3d, prereg 77f734c.
2. **Правило-рычаг (аналитически).** Локальное линейное считывание f₂
   скрытого вектора ⇒ E=cosθ/3 точно (вывод выше). Сессия C3-L,
   battery_c3l_l2.py; коммит **49e8c1b**.
3. **L2 frame-local space-like (эта кампания).** «Каждый первый» без
   совместного закона ⇒ факторизованный LHV: E(0)=0.333, E(90)=0,
   E(180)=−0.333, S=0.943. Коммит **49e8c1b** (C3L_L2.json).

## Значение для программы
Три разные идеи (диссипативная динамика, аналитический рычаг правила,
frame-local разрешение) сходятся в cosθ/3 ⇒ это НЕ артефакт одной
конструкции, а структурная граница локально-линейного слоя. Верхний
берег шва — 2√2 (совместный хордовый). Расстояние 1/3 → 2√2 = амплитудный
шов, центральная задача C3-S. «Стена цикла 2» (реляксация не достаёт
квантовых корреляций) и «шов реконструкции» (shared-λ не достаёт 2√2) —
одно и то же, с двух сторон.
