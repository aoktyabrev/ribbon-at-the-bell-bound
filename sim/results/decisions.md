## 2026-07-06 — Фаза B, R1: несходимость + ферро-знак, СТОП перед R2

**Что произошло.** R1 (κ ∈ {0.1, 1, 10}, geodesic, T=0) прогнан по пре-регистрации.
Сырой результат при 6000→12000 шагах: κ=10 дал форму +cosθ (по форме p≈2), ВОПРЕКИ
пре-регистрации (ожидались пила/p≈1, p=2 не ожидался). Знак корреляции —
ферромагнитный: E(0)=+1 (концы выравниваются), а не −1 (синглет).

**Паранойя (SPEC §9.2, обязательна при p≈2).** Скрипты `paranoia_R1.py`,
`conv_study.py` → `results/R1/paranoia.md`, `results/R1/convergence_study.md`:
1. E(θ) при κ=10 ДРЕЙФУЕТ с числом шагов (|ΔE| 6k→24k = 0.35), круче к ступени.
2. Удвоение N,B меняет E на 0.58 (длинная цепь релаксирует медленнее — недосходимость).
3. Знак ферро — не баг: при θ=0 доля s==t = 0.975; классификатор верен.
4. Истинное равновесие: κ=1 сходится (~30k шагов) к неполной ферро-корреляции
   E≈0.38·cosθ (НЕ насыщается до ±1 ⇒ вне семейства хорды); κ=10 сходится (>100k
   шагов) к ферро-СТУПЕНИ при 90° (p→∞, PR-box). **p=2 нигде; это транзиент.**

**Вывод.** Пре-регистрированный энергетический критерий сходимости (|ΔE|/шаг<1e-6,
однократное удвоение) неадекватен: недостижим при k_e~сотни и допускает лишь 6k→12k
шагов, тогда как κ=10 требует >100k. Приведённые в `report.md` кривы R1 суть
транзиенты неполной релаксации; показатель p̂≈1.93 для κ=10 — артефакт.

**Решения (требуют подтверждения архитектора):**
- (анализ, применено) Фит закона хорды теперь ПЕРЕБИРАЕТ знак корреляции
  (`analysis.fit_chord_signed`): пре-регистрированный фит молча предполагал
  антиферро (E(0)=−1) и вырождался на границу сетки на ферро-данных. Это
  исправление вырождения, НЕ подгонка p (показатель падает куда падает). Пре-
  регистрация в yaml не тронута.
- (протокол, СТОП) R2 (термический) НЕ запущен: гонять отжиг поверх несошедшегося
  и неверно-понятого базиса = наука на битом конфиге (CLAUDE.md запрещает).
- Физика (для архитектора, НЕ интерпретирую сам): изотропная упругая лента фазы 1 —
  ФЕРРОМАГНЕТИК (выравнивает концы), а не синглет. Семейство хорды к ней неприменимо
  (κ=1 не насыщается; κ=10 → ступень). Возможные направления — анти-зажим,
  ограничение нулевой скрутки (R4) или спинорный режим (R5) для получения синглета.

**ARCH-QUESTION (см. сводку архитектору):** критерий сходимости и режим шагов;
ферро- vs синглетная постановка; смысл R2 до сходимости R1; фикс κ=0.1 (маргиналы
отклонены на ~3.6σ, устойчиво к steps — мелкий отдельный вопрос).

## 2026-07-06 — Решение архитектора: R5 ВНЕ ОЧЕРЕДИ (до R1b/R2)

Обоснование архитектора: критический тест спинорной структуры (различение q/−q,
720°-периодичность) важнее упорядоченного прохода. R2/R3/R4 не трогать.

**Внедрено по указанию:**
1. `energy.py`: третий режим `elastic: spinor` — геодезия БЕЗ модуля
   d(p,q)=arccos(clip(<p,q>)), различает q и −q. Тест: d(q,−q)=π (spinor) vs 0 (geodesic).
2. Новый критерий сходимости (ответ на ARCH-Q#1): БЛОЧНЫЙ по НАБЛЮДАЕМОЙ —
   `max_θ |ΔE| < max(2·σ_MC, 0.01)` между блоками, потолок 200k шагов
   (`dynamics.block_convergence`). Заменяет неадекватный энергетический |ΔE|<1e-6.
3. Диагностика кинков: счётчик пар <q_i,q_{i+1}> < 0 в финале (полуоборотные кинки
   на S³); плотность по θ/ячейкам + корреляция «кинки ↔ ветвь» (гипотеза: миноритарная
   ветвь несёт больше кинков).
4. R5 = 2×2 ячейки (elastic∈{geodesic,spinor} × κ∈{1,10}) для честного сравнения под
   ОДНИМ блочным протоколом. Пре-регистрация — `experiments/R5.yaml` (до запуска):
   (a) знак остаётся ферро; (b) кинки застревают при T=0, плотность>0;
   (c) амплитуда E(0) при κ=1 spinor НИЖЕ geodesic; (d) A3/A1 — открытый вопрос.

**Параллельно (дёшево):** R1b (свип N∈{2,4,8,16,32,64}, κ=1, geodesic, до сходимости —
гипотеза падения амплитуды 0.62→0.38); диагностика κ=0.1 (зеркало a→−a, антитетическая
инициализация n/−n, гистограмма |n_A·a|). Скрипты `r1b_Nsweep.py`, `margin_diag.py`.

После R5 — стоп, сводка архитектору.

### R3 — КРАСНЫЙ КОНТРОЛЬ (стоп по протоколу)

R3 (Коссера, T=0, блочный) прогнан (4077с). **Физика драматична:** жёсткость ИЗГИБА
управляет ножницами — kt/kb=0.1 (жёсткий изгиб) → ступень амп=1.0; kt/kb=1 → 0.82;
kt/kb=10 (мягкий изгиб) → декуплирование амп=0.01. НО **контроли §4.2 КРАСНЫЕ:**
±-симметрия нарушена до 127σ (kt/kb=0.1), 92σ (изотроп); воспроизводимость падает
(19σ, 8σ) у жёстких ячеек; мягкая (kt/kb=10) почти чиста.

**Диагноз (не физика, численность):** нарушение ±-симметрии сосредоточено при θ=0
(осевыровненный зажим a=b=e_z), |P(pp)−P(mm)|≈0.67, СОГЛАСОВАННО по обоим сидам (не
метастабильность). Энергия Коссера ПРОВЕРЕНА ±-симметричной (инвариантна к флипу осей
q→q⊗j, как и geodesic). ⇒ симметричная энергия + симметричная Haar-инициализация →
асимметричный итог возможен только через УСИЛЕНИЕ float32-смещения нестабильной
динамикой у симметричного седла θ=0. Худшее у самых ЖЁСТКИХ/долгих релаксаций (200k
шагов), почти нет у мягкой — сигнатура совпадает с артефактом κ=0.1 (float32-анизотропия
осевыровненной сетки). Подтверждение — тест поворота зажимов (r3_diag.py, в работе):
поворот прочь от осей должен убить асимметрию.

**СТОП (протокол архитектора: стоп после красного контроля).** R4/R4a/R4b/мягкая лента
НЕ запускаются. Знак ферро подтверждён; анизотропия изгиба сильно двигает ножницы —
но численный артефакт при осевыровненных зажимах мешает чистому измерению. Вопрос к
архитектору: (i) повернуть зажимы прочь от осей для всех прогонов (общий фикс θ=0-седла);
(ii) отжиг вместо T=0-GD для выхода из float32-смещённого седла; (iii) float64 (но fp64
на 4070Ti нереалистичен). Пре-регистрация R3 не тронута.

- **R4 (Tw=const, после зелёного R3):** проекция градиента g←g−(g·∇Tw/|∇Tw|²)∇Tw,
  Tw=Στ_i. Тест |ΔTw|<1e-4. R4a: сектор Tw=0, спинор, свип θ, M1/M2 (антиферро?
  центральный вопрос, обе гипотезы легальны). R4b: Tw=0 vs Tw=2π бок о бок (ℤ₂), зависит
  ли E(θ) от сектора. Мягкая лента: спинор, k_e∈{1,3}≈T0, decay=0.9995, h-популяции,
  E(θ|h), M2 — P(h=−1)>0 впервые, расщепление E по h открыто.
