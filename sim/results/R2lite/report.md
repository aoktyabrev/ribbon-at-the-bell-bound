# R2lite — отчёт (full)

**Описание:** Изотропная лента (geodesic), κ=1, N=64. Ланжевен с геометрическим отжигом, T0=1.0, три скорости decay. Полный свип θ, фиты семейств + гармоники A3/A1. Меняет ли термическая мера форму кривой относительно R1 (T=0).


**Пре-регистрированное предсказание:** Отжиг ПОДНИМАЕТ амплитуду (выбивает ленты из локальных минимумов), но ведёт форму к СТУПЕНИ/tanh, а НЕ к косинусу; A3/A1 уходит в минус с замедлением отжига. Знак остаётся ферро. (Согласуется с превью из R5b: 0.49→0.66→0.99 по decay.)

## Вердикт против пре-регистрации

Предсказание: Отжиг ПОДНИМАЕТ амплитуду (выбивает ленты из локальных минимумов), но ведёт форму к СТУПЕНИ/tanh, а НЕ к косинусу; A3/A1 уходит в минус с замедлением отжига. Знак остаётся ферро. (Согласуется с превью из R5b: 0.49→0.66→0.99 по decay.)


- Амплитуды max|E| по скоростям отжига: 0.999:0.496, 0.9995:0.661, 0.9999:0.989.
- Лучшие модели: 0.999:хорда-p, 0.9995:хорда-p, 0.9999:хорда-p.
- Проверка гипотезы «термическая мера сглаживает кривую, форма семейства не меняется» — сопоставить амплитуды и лучшие модели с R1 (см. сводку).

> **Вердикт (гипотеза архитектора §4):** амплитуда с замедлением отжига **РАСТЁТ** (0.496 → 0.661 → 0.989); A1(cosθ) +0.49 → +0.63 → +1.02, при decay=0.9999 |E(0)|→0.99 ⇒ форма ведёт **к СТУПЕНИ, а НЕ к косинусу** (A3 остаётся ≈0). Знак ферро во всех ячейках ✅. **Гипотеза подтверждена:** изотропная лента при T>0 не даёт квантовую косинусную форму — «ножницы» §4 (|E(0)|→1 только через ступень). Согласуется с R5b (спинор): geodesic и spinor термически неотличимы.
