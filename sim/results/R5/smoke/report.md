# R5 — отчёт (smoke): спинорная лента vs geodesic

**Описание:** Спинорная лента vs изотропная (geodesic). elastic∈{geodesic, spinor} × κ∈{1,10}. Спинорный режим штрафует полуоборотные кинки <q_i,q_{i+1}> < 0, которые при T=0 застревают. Вопрос: сдвигает ли спинорная структура форму/амплитуду и как ведут себя кинки относительно ветви исхода.


**Пре-регистрация:** (a) знак остаётся ферро (n(−q)=n(q), зажимы знако-слепы); (b) кинки застревают при T=0, плотность > 0; (c) амплитуда E(0) при κ=1 spinor НИЖЕ, чем geodesic (ловушки кинков подавляют выравнивание); (d) форма A3/A1 — без прогноза, открытый вопрос.

## Диагностика кинков ↔ ветвь исхода

Гипотеза (пре-рег b): ленты в МИНОРИТАРНОЙ ветви несут больше кинков. Превышение = ⟨кинков|минор.ветвь⟩ − ⟨кинков|мажор.ветвь⟩ (на ленту, из 63 связей), усреднено по θ.

| geo κ=10 | 0.5034 | -0.701 |
| spin κ=1 | 0.0000 | +0.000 |
| spin κ=10 | 0.0000 | +0.000 |

**(a) знак ферро** (n(−q)=n(q), зажимы знако-слепы): ✅ подтверждено (все ячейки ферро).
