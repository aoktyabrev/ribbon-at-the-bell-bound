# R5 — отчёт (full): спинорная лента vs geodesic

**Описание:** Спинорная лента vs изотропная (geodesic). elastic∈{geodesic, spinor} × κ∈{1,10}. Спинорный режим штрафует полуоборотные кинки <q_i,q_{i+1}> < 0, которые при T=0 застревают. Вопрос: сдвигает ли спинорная структура форму/амплитуду и как ведут себя кинки относительно ветви исхода.


**Пре-регистрация:** (a) знак остаётся ферро (n(−q)=n(q), зажимы знако-слепы); (b) кинки застревают при T=0, плотность > 0; (c) амплитуда E(0) при κ=1 spinor НИЖЕ, чем geodesic (ловушки кинков подавляют выравнивание); (d) форма A3/A1 — без прогноза, открытый вопрос.

## Вердикт по пре-регистрации

**(a) знак ферро** (n(−q)=n(q), зажимы знако-слепы): ✅ подтверждено (все ячейки ферро).

**(b) кинки застревают при T=0, плотность>0**: ❌ (spinor плотности: 0.000, 0.000).

**(c) амплитуда κ=1: spinor < geodesic**: НЕОДНОЗНАЧНО. По max|E| — равны (geo=0.379,
spin=0.378, разница в шуме). По E(0) — spin ниже (0.358 vs 0.379, ~2.9σ, слабый реальный
эффект). НО пре-регистрированный МЕХАНИЗМ (ловушки кинков подавляют выравнивание)
ОПРОВЕРГНУТ пунктом (b): кинки не застревают (плотность spinor = 0). Слабое подавление
E(0) — не от кинков; природа — вопрос к архитектору.

**(d) форма A3/A1** — без прогноза, открытый вопрос. Значения:
  - geo κ=1: A3/A1 = -0.032 (A1=+0.386, A3=-0.013)
  - geo κ=10: A3/A1 = -0.255 (A1=+1.243, A3=-0.317)
  - spin κ=1: A3/A1 = -0.016 (A1=+0.385, A3=-0.006)
  - spin κ=10: A3/A1 = -0.254 (A1=+1.243, A3=-0.316)
