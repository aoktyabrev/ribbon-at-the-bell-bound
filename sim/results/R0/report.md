# R0 — отчёт (full)

**Описание:** Разрезанная лента: k_e = 0, связь между концами отсутствует. Каждый конец — независимый кадр под квадратичным зажимом с двумя минимумами ±ось. Контроль исправности зажима, классификатора ветвей и динамики.


**Пре-регистрированное предсказание:** E(θ) ≈ 0 для всех θ (независимые концы ⇒ ⟨s·t⟩ = ⟨s⟩⟨t⟩ ≈ 0); маргиналы P(s=+) = P(t=+) = 0.5 ± 3σ; ±-симметрия P(s,t) = P(−s,−t). Если нет — баг, стоп.

## Контроли (SPEC §4.2)

- **Маргиналы 0.5 ± 3σ:** ✅ прошли (все P(s=+),P(t=+) в пределах 3σ от 0.5).
- **±-симметрия P(s,t)=P(−s,−t):** ✅ прошла (max |Δ| = 2.27σ).
- **Воспроизводимость на 2 сидах:** ✅ max|ΔE| = 0.0635 (2.87σ).

**Гармоники E(θ):** c0=-0.0025, A1(cosθ)=-0.0031, A3(cos3θ)=-0.0058.

## Вердикт

Предсказание R0: E(θ) ≈ 0 всюду, маргиналы 0.5.
- max|E(θ)| = **0.0264** при пороге 3σ = 0.0331 → в пределах шума ✅.

> **R0 ПРОЙДЕН.** Разрезанная лента (k_e=0) даёт нулевую корреляцию и симметричные маргиналы — классификатор ветвей и зажимы исправны.
