# Фаза C — сводный отчёт архитектора

**Проект quantum_entanglement / ribbon-sim. Июль 2026.**
**Статус: фаза C закрыта. Все режимы протестированы, механизм найден,
квантовая амплитуда классической мерой не достигнута, предел назван.**

---

## 1. Главный результат в одну страницу

Топологический ℤ₂-заряд связи (сектор Tw = 2π классической упругой ленты)
воспроизводимо наследует **знак** и **синглетную форму** корреляции через
энтропийную меру бассейнов:

    сектор Tw = 0:   E(θ) — ферро-ступень, E(0) = +1
    сектор Tw = 2π:  E(θ) = −A·cos θ,  A ≈ 0.5,  A3/A1 ≈ 0.04

Механизм измерен точно (R4d): E(θ|2π) = −tanh(ΔF/2T), где ΔF ∝ T, то есть
ΔE ≈ 0 и вся асимметрия ветвей — **энтропийная**: ΔS ≈ 1.1·cos θ.
Антиферро-ветвь выигрывает не глубиной минимумов, а их числом (~e^1.1 ≈ 3×).

**Программа заметки §6 реализована буквально:** вероятность ветви =
мера множества микроконфигураций, релаксирующих в ветвь, — и этот подсчёт
сам, без вкладывания руками, рождает косинусную форму синглета. Форма
косинуса возникла при заблокированной сверке с эталонами и разгромила
альтернативы по AIC (синглет 21.7 против пилы 62, ступени 279, хорды 507).

**Но квантовая амплитуда не достигается:** A = tanh(ΔS/2) ≈ 0.5;
CHSH = A·2√2 ≈ 1.48 < 2 — кривая внутри классического предела. Рост A
с охлаждением логарифмически-медленный (0.29 → 0.52 за две декады T).

## 2. Стена, названная по имени: точные нули

Квантовая механика требует P(одинаково | θ=0) = 0 — не подавления, а
точного нуля (суперселекция от сохранения полного спина: запрет наложен
на ИСХОД). Энтропийный подсчёт конечной классической системы точных нулей
не даёт по определению: больцмановский вес давит ветвь, но не обнуляет.
A = 1 требует ΔS = ∞, т.е. строго нулевой меры ферро-ветви.

Наша связь Tw = const — точная, но на КОНФИГУРАЦИЯХ; квантовое «полный
спин = 0» — точная связь на СОВМЕСТНОМ ИСХОДЕ. Зазор между этими двумя
типами ограничений и есть зазор между A = 0.5 и A = 1.

**Формулировка недостающего ингредиента (самая точная за проект):
правилу Борна нужны точные запреты ветвей; классическая энтропийная мера
даёт только штрафы.**

## 3. Лемма-кандидат (черновик; закрывается результатом R4e)

Для классической системы с термо-мерой бассейнов и конечной энтропийной
асимметрией ветвей ΔS(θ):

    E(θ) = −tanh(ΔS(θ)/2),  |E| < 1 при ΔS < ∞.

(i) Если ΔS не зависит от размера системы (интенсивна, задаётся зарядом) —
амплитуда топологически заперта: синглетная форма достижима только с
классической амплитудой, CHSH < 2 навсегда.
(ii) Если ΔS ∝ N (экстенсивна) — амплитуда управляема длиной, но форма
обязана уйти от косинуса (tanh насыщается: середина уплощается к ступени,
|A3/A1| растёт). «Ножницы форма–амплитуда» становятся законом: cos θ и
|E(0)| = 1 несовместимы в классе энтропийных мер.
В обеих ветвях правило Борна (форма cos И амплитуда 1 одновременно)
недостижимо. Выбор ветви — открыт до аудита R4e (экстенсивность ΔS(N)).

## 4. Полное отрицательное множество (что НЕ рождает квантовую меру)

Пять механизмов × пять формулировок энергии, все с зеркальными контролями
и аудитами сходимости:

1. Изотропная упругость, T=0 и отжиг (R1, R2): ферро-линия «пологий
   косинус ↔ ступень», ножницы амплитуда–форма.
2. Спинорная метрика сама по себе (R5, R5b): гейдж-слепота осевых
   наблюдаемых (no-go: SU(2)→SO(3)); кинки без топозащиты уходят, при
   T0 ≪ k_e не населяются. Мягкая лента (k_e ≈ T0): кинки всё равно
   вымерзают; заряженный сектор термически недостижим — только явное
   приготовление.
3. Анизотропия изгиб/скрутка (R3): выравнивание переносит изгиб; скрутка
   в ферро-механизме не участвует.
4. Связь Tw=0 (R4a): транзиенты меняются, равновесие осевых наблюдаемых —
   нет (динамический no-go).
5. Заряженный сектор Tw=2π (R4b–R4d): МЕХАНИЗМ НАЙДЕН (знак + форма),
   амплитуда ограничена энтропийно.

Побочные твёрдые результаты: динамический интерьер (амплитуда 0.66→0.38,
насыщение N≥16 — бассейны определяются динамикой, не ландшафтом);
вырожденность основного состояния фрустрированного сектора при T=0
(DEGENERATE-класс: наблюдаемая не самоусредняется); глассовость сектора
2π под отжигом и её разрешение параллельным темперированием.

## 5. Методология (что сделало результаты твёрдыми)

Пре-регистрация всех прогонов; блочная сходимость по наблюдаемой;
зеркальные пары; честный сэмплер секторов; PT для глассовых ландшафтов;
правило «сверка с эталоном после фиксации сырых весов».

Дисциплина аудита исправила по ходу три содержательные гипотезы:
«p≈2» в R1 (транзиент), диагноз float32 в R3 (дефект формулировки,
опровергнут пре-регистрированным мини-fp64), «энергетический tanh» в R4d
(механизм оказался энтропийным). Две из трёх — гипотезы архитектора.
Плюс отзыв предсказания «пила при κ=10» до вскрытия данных (теорема
жёсткого предела). Ни один из главных выводов не опирается на неаудированный
прогон.

## 6. Дверь в фазу D

Вопрос фазы D формулируется из стены §2: **может ли топология вложения
превратить энтропийное подавление в точный запрет?** Конкретно: настоящая
двумерная поверхность, вложенная в R⁴; 3D-срез с двумя пересечениями
(«частицы»); зажимы действуют в срезе; классы оснащения π₁(SO(3)) = ℤ₂.
Гипотеза для проверки: существует ли постановка, в которой выровненный
исход при θ=0 в нетривиальном классе запрещён геометрически (нулевая мера
по построению), а не подавлен энтропийно. Также в фазу D переходит
PBR-ограничение на меру (носители мер разных состояний не пересекаются) —
готовый тест любой будущей конструкции.

## 7. Финальная редакция леммы (после R4e-v2) — фаза C закрыта

R4e-v2 отверг и экстенсивность, и const, и коммензурабельность (четвёртая
исправленная гипотеза архитектора): истинная структура — кроссовер по
плотности твиста. Заряд 2π локализован твист-солитоном в середине ленты
(пик ⟨τ̃⟩ ≈ 0.9, контр-твист у зажимов); ферро при N−1 ≤ 33, антиферро-окно
~48–64, гашение при N−1 ≥ 95. Синглетный результат R4c/R4d реален, но
конечно-размерен: при N → ∞ твист разбавляется и сектор локально невидим.

Лемма, окончательно: классическая энтропийная мера с топологическим
зарядом даёт синглетную форму E = −tanh(ΔS·cosθ/2), но (i) без точных
нулей (A = tanh(ΔS/2) < 1 всегда) и (ii) без масштабной инвариантности
(эффект гаснет с длиной). Квантовой корреляции присущи обе инвариантности;
классическому механизму — ни одна. Борн недостижим ни при каком N.

Фаза D наследует два точных вопроса: может ли топология вложения в R⁴
(1) превратить энтропийное подавление в геометрический запрет ветви и
(2) сделать эффект заряда инвариантным к растяжению. Оба свойства —
определяющие черты истинных топологических инвариантов, что и составляет
исходную интуицию модели («4D-объект, наблюдаемый 3D-срезом»).
Опционально запарковано: доперемешивание N−1 = 40–56 (уточнение пика окна).
